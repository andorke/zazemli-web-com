# site-shell — delta (guide-lab)

## MODIFIED Requirements

### Requirement: Заглушки без выдуманной копи
Страница `/diary-signup` SHALL рендериться как заглушка без заголовка-выдумки в DS-стиле; произвольные маркетинговые тексты MUST NOT добавляться до получения копи и прототипа (юр-переделка формы на два чекбокса — 152-ФЗ + 38-ФЗ — pending на стороне vault). `/lab` и `/guide` заглушками больше не являются.

#### Scenario: Последняя заглушка
- **WHEN** открыта `/diary-signup`
- **THEN** страница рендерится заглушкой без выдуманной копи, с noindex

#### Scenario: Контентные страницы не заглушки
- **WHEN** открыты `/lab` и `/guide`
- **THEN** обе содержат полный контент из своих спецификаций (lab-page, guide-page)
