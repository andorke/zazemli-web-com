# landing-redesign

## Why

Эталон главной сменился: вместо Figma 185:2 (9 секций) — HTML-прототип `../zazemli-vault/Айти/Сайт/prototypes/landing.html` (11 контент-блоков) + канон-копи `home.md` v2.1. Появились новые блоки доверия («Как это работает», мост «почему именно так → /lab»), манифест, атмосферный баннер; секция Statement удалена; галерея получила карточку-приглашение «N° 08 — ?» и стала вести на страницы товара. Без перевёрстки главная остаётся в старой типографике и структуре, не соответствующей канону.

## What Changes

- **BREAKING**: структура главной — 9 секций Figma → 11 блоков прототипа: Hero → Манифест → «Как это работает» (3 шага) → Галерея `#collectio` → Колбы (DifferentSoil) → «Что в боксе» → Атмосферный баннер → «Что даёт» → «О нас» → Тизеры ×3 → Ozon-CTA. Statement («Выращивай и создавай простое») удаляется.
- Тексты: канон `home.md` v2.1 (Hero H1 «Заземли растение. Заземли себя.»); для блоков, которых нет в каноне («Как это работает», баннер), — тексты прототипа. Порядок блоков — по прототипу (в каноне «что в боксе» и колбы переставлены — расхождение, фиксируется в вопросах Насте).
- Галерея: 7 карточек (фото-слот, «N° 01»-нумерация, имя) + 8-я карточка-приглашение «N° 08 — ?» + CTA «Вся коллекция →»; карточки кликабельны → `/collectio/[slug]` (появятся в change `product-pages`; до него — ссылки на якорь).
- Мост доверия: в секции колб ссылка «почему именно так →» на `/lab` цветом moss-ink.
- Тизеры: три (Лаборатория → `/lab`, Гайд → `/guide`, Дневник — **без ссылки**, инвариант «diary вне навигации» сохраняется).
- Контент `src/content/home.ts` пересобирается под новые блоки; инварианты (SKU=7) сохраняются.
