# site-shell — delta (diary-signup-form)

## MODIFIED Requirements

### Requirement: Заглушки без выдуманной копи
Страницы `/collectio`, `/lab`, `/guide` SHALL рендериться как заглушки: kicker/заголовок из утверждённых названий (Коллекция, Лаборатория, Гайд) в DS-стиле. Произвольные маркетинговые тексты MUST NOT добавляться до получения копи. `/diary-signup` из списка заглушек исключается — реализована формой (capability `diary-signup`), при этом сохраняет `noindex` и отсутствие в навигации.

#### Scenario: Заглушка рендерится
- **WHEN** открыта `/lab`
- **THEN** страница показывает заголовок «Лаборатория» в DS-типографике и не содержит выдуманных описаний

#### Scenario: diary-signup — форма, не заглушка
- **WHEN** открыта `/diary-signup`
- **THEN** страница показывает форму подписки по прототипу (не заглушку), сохраняя `noindex` и отсутствие в шапке/футере
