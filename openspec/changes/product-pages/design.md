# product-pages — design

## Context

7 прототипов `collectio-*.html` (единый шаблон, различия — данные SKU), канон `product-description.md` v1.1.1 (8 блоков, таблица биотопов, A/B hero), `spotlight-quotes-7-sku.md` (ритуал-фразы). Роут `/collectio` сейчас — заглушка; QR партии 0 указывает на него. Полный контекст — `docs/superpowers/specs/2026-07-05-site-redesign-design.md`.

## Goals / Non-Goals

**Goals:**
- 7 страниц товара со static export, единый шаблон, данные из типизированного `sku.ts`.
- Рабочий клиентский редирект `/collectio` → `/#collectio` (QR не ломается).
- Клик-путь главная → карточка → страница товара → Ozon-CTA замкнут (Ozon — в состоянии «Скоро», пока URL нет).

**Non-Goals:**
- Реальные фото (слоты), Ozon-ссылки (ozonStoreUrl: null), корзина/оплата (продажа — на Ozon).
- Отдельная страница-индекс коллекции (решение канона: коллекция на главной).

## Decisions

1. **Шаблон один, данные в `sku.ts`.** Страница — `app/collectio/[slug]/page.tsx` + `generateStaticParams` по 7 slug. Slug = латинские имена из прототипов (`collectio-monstera.html` → `monstera`). Альтернатива «7 статических page.tsx» отклонена: дублирование шаблона.
2. **Типы данных SKU расширяются полями**: `vial` (проценты основа/воздух/влага/дренаж), `composition[]` (компонент + материал-цвет + доля), `boxContents[]`, `care`, `ritualPhrase`, `sizes[]` (объём + цена + ozonListingUrl: null), `sourceNote` (claim + источник). Источник значений: `product-description.md` v1.1.1 (+ таблица биотопов), при пробелах — прототип соответствующего SKU. Расхождения — в вопросы Насте.
3. **Редирект `/collectio`** — `page.tsx` рендерит пустую страницу с `<meta http-equiv="refresh" content="0;url=/#collectio">` + инлайн `location.replace('/#collectio')` + `<link rel="canonical" href="/">` + noindex. Не `redirect()` из next (не работает в static export), не отдельный HTML в public (теряет layout-метаданные). После выбора хостинга поверх добавится настоящий 301 (вне скоупа).
4. **Размер-селектор — клиентский компонент** (`'use client'`): переключает объём/цену/текст CTA «Купить {размер} на Ozon». При `ozonListingUrl: null` кнопка — состояние «Скоро на Ozon» (не ссылка), UTM — через существующий `lib/utm.ts` (`utm_content=sku00X`).
5. **SourceNote** — details-аккордеон (атом ds-migration) с бейджем «рецензируемо» и источником (автор, год, издание); полный реестр источников — на /lab, со страницы товара один мост «Весь состав и источники → в лаборатории».
6. **SKU-цвет как CSS-переменная `--sku` на корне страницы**; применяется только в декоративных местах (kicker-акцент, бордер дневник-панели, RitualNote, MaterialDot-заливки состава). Тест: один SKU-цвет на страницу (политика tokens v1.1.0).
7. **Founder-quote** — новая редакция «…а руками тянется к земле» (прототипы; канон-патч product-description pending на стороне vault — исключение №2 дизайн-дока).

## Risks / Trade-offs

- [Клиентский редирект медленнее серверного 301 и не передаёт SEO-вес] → приемлемо: `/collectio` носитель QR-трафика, не SEO; canonical + noindex сглаживают; 301 добавится конфигом хостинга.
- [Данные 7 SKU объёмны, риск опечаток в %/ценах] → unit-тест инвариантов: сумма % колбы = 100, цены > 0, slug уникальны, ровно 7 SKU.
- [A/B-варианты hero в каноне (Настя выбирает в Figma)] → берём вариант A; выбор — в вопросы Насте.

## Migration Plan

Роут и данные добавляются аддитивно; заглушка `/collectio` заменяется в последнюю очередь вместе с переключением карточек галереи. Откат — git revert.
