## 1. Что это
ЗАЗЕМЛИ — российский D2C-бренд боксов для пересадки комнатных растений. Сайт `zazemli.com` = витрина доверия и бесплатные магниты; **единственная конверсия — внешняя ссылка на Ozon** (сайт сам не продаёт). Тон: тихо, дорого, музейно, дружелюбное комьюнити.

## 4. Стек (канон проекта)
- **Next.js + React + Tailwind CSS + shadcn/ui.** Компонентные либы (shadcn / 21st / Magic UI) ставятся в репозиторий, НЕ как плагины.
- **CMS: Sanity** (`sanity-plugin` установлен). На MVP контент может быть и статикой/MDX — согласовать с Настей, что заводить в Sanity сейчас, что позже.
- **Анимация:** Концепт A+ — **только CSS + ванильный JS** (IntersectionObserver для scroll-reveal), без Framer Motion / GSAP / Lenis / WebGL. `prefers-reduced-motion: reduce` — обязателен (глобально отключает анимацию). Демо и спека: `_restored/PHASE8_MOTION_concept_2026-06-11.md` + `PHASE8_MOTION_demo_2026-06-11.html` (открой в браузере — там готовые CSS-приёмы: fade-up, перелистка /guide, аккордеон /lab). NB: тайминги в демо 700–900 мс «тихо-дорого» vs канон A ≤500 — финальный выбор за Настей.
- **Хостинг / домен / почта `team@zazemli.com`** — твоя зона (ТРЕК 4). Уточни у Насты текущий статус домена и Ozon-store URL.
