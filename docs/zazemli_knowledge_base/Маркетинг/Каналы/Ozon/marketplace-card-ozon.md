## Что это

Универсальный шаблон карточки маркетплейса (Ozon + Я.Маркет) для 16 SKU = 7 растений × 2-3 размера. Расширяет `marketplace-card-ozon.md` (Ozon-профиль) добавлением Я.Маркет-профиля и универсализацией под все 16 SKU. Главный коммерческий артефакт запуска (70-80% выручки по плану года 1).

После Спринта 7 этот файл — single source of truth для маркетплейс-карточек; `marketplace-card-ozon.md` становится ссылкой-указателем (см. §«Заметка про дубль»).

## Роль в ДНК + канал обучения

- **3 канала × тон обучения:** маркетплейс = переводим в формат «3-5 секунд на первый экран», не учим. См. `register-matrix.md`.
- **Доминанты ДНК для жанра:** Фармацевт 50% (плотный технический текст, точные цифры) + Натуралист 30% (биотоп как опора рецептуры) + Галерист 15% (типографическая чистота, без декора) + Друг 5% (одна тёплая нота в B5 — без сюсюканья).
- **Регистр:** Наставник. Не Редактор (нет места поэтике), не Подруга (маркетплейс не диалоговый).

## JTBD-тройка

- **Emotional:** доверять выбору — рецептура под конкретное растение, а не универсальная земля.
- **Social:** иметь авторский российский бренд в каталоге, не «ещё один пакет HURY».

## Состав / структура текста

### Два профиля площадки

#### A. Ozon-карточка

| Слот | Лимит | Что внутри |
|---|---|---|
| Title | ≤200 знаков индексируется, оптимум до 90 | формула (см. ниже) |
| 5 буллетов | 180 / 220 / 200 / 200 / 180 | B1–B5 (см. иерархию) |
| Видеообложка | 15 сек, 9:16 | медальон с N° |
| Характеристики | таблица по категории | заполнять ВСЕ поля |
| Rich-content | ≤5000 знаков | расширенное описание для лаб-блока (опц) |

**Формула Title (Ozon):**

```
Грунт для {растение_рус} под горшок {объём} л · ЗАЗЕМЛИ
```

- Главный SEO-ключ «грунт для {растение}» — первые 30 символов.
- «под горшок {объём} л» — отстройка от HURY (они продают по литражу пакета, мы — по объёму горшка). Заодно второй атрибут для алгоритма.
- «ЗАЗЕМЛИ» — последним, после ключа+объёма (бренд-имя обязательно по правилу sender naming, см. `brand-voice-profile` v1.0.2 §«Sender naming»).
- N° **не** в title (см. `sku-system.md` §1 «Где НЕ применяется»).

**Иерархия 5 буллетов (Ozon):**

- **B1 (≤180):** что это. Тип + ключ + объём + растение одной фразой. Главный SEO-узел.
- **B2 (≤220):** состав. 3-5 ключевых компонентов из 11, pH, аэрация. **Тон Founder Quote 3** — конкретика по растению, противопоставление «не торф — Y». Источник готовых формулировок — `audit-bz-copy-library.md` §4.1 (11 компонентов).
- **B3 (≤200):** использование. Возраст растения, объём горшка, частота пересадки (3 шага). **Формулировки из Founder Quote 3:** «корневин — присыпать корни для поддержки», «дренаж с запасом под разные формфакторы».
- **B4 (≤200):** brand-statement. Дневник, ритуал, коллекция. Slogan Lvl 1-2 (см. `slogan-matrix.md`).
- **B5 (≤180):** confidence-statement. Научный moat (80+ источников) + кто делает. Caveat = 0.

#### B. Я.Маркет-карточка

| Слот | Лимит | Что внутри |
|---|---|---|
| Заголовок | ≤150, оптимум 60-120 | формула (см. ниже) |
| **Категория** | — | **Дом / Сад / Огород / Грунт и удобрения / Грунт для комнатных растений** (подтверждено решением 2) |
| Характеристики | ~50-100 атрибутов | **обязательно заполнить все** — даёт больше веса, чем текст |
| Аннотация | 400-600 знаков (рекомендация Я.Маркета) | 4-5 коротких абзацев из B1-B5 |
| Изображения | 3:4 | те же фото, что Ozon, ресайз |

**Формула заголовка (Я.Маркет):**

```
Грунт для {растение_рус} ЗАЗЕМЛИ — отдельная смесь под него, {объём} л
```

- Бренд раньше (Я.Маркет ценит производителя выше Ozon).
- «Отдельная смесь под него» — USP в заголовке, на языке покупателя. **Запрещено** «рецептура под род» (правило Насты 2026-05-25 — «род» это биологический термин, не в обиходе).

**Отличие от Ozon:** аннотация — сплошной текст 5 коротких абзацев (а не 5 буллетов). Атрибуты в Я.Маркете дают больше веса, чем красивая аннотация, — поэтому **сначала заполняем все характеристики, потом пишем текст**.

## Шаблон + 3 примера

### Пример 1 · N°001 Монстера 3.5 л (Ozon)

**Title (45 знаков):**

```
Грунт для монстеры под горшок 3.5 л · ЗАЗЕМЛИ
```

**B1 (148 знаков):**

```
Грунт для монстеры, на горшок 3.5 л. Рецептура под лиану облачного леса: двойная структура — подстилка для проростка и аэрация для взрослой.
```

**B2 (215 знаков) — Founder Quote 3 tone:**

```
11 компонентов под монстеру: торф нейтральный, кора, кокос, мох сфагнум, цеолит, биочар. Монстере нужно удержать влагу — поэтому мох, а не пеностекло, как Замиокулькасу. pH 5,5–6,0.
```

**B3 (198 знаков) — Founder Quote 3 tone:**

```
Объём 3.5 л — на горшок 18–20 см. Раз в год, весной. Дренаж керамзитом — с запасом под разные формфакторы горшков. Корневин 5 г — присыпать корни для поддержки после пересадки.
```

**B4 (198 знаков):**

```
В коробке — почвосмесь, дренаж, корневин 5 г, нитриловые перчатки, листовка с гайдом по пересадке и дневник растения на год. Ритуал пересадки — собранный, без беготни по «всё для сада» магазинам.
```

**B5 (172 знака):**

```
Рецептура построена на 80+ рецензируемых источниках: Aroid Society, UF/IFAS, Missouri Botanical Garden. Собираем небольшими партиями в Москве. Сайт лаборатории — zazemli.com
```

**Видеообложка:** медальон с **N°001** + дудл Монстеры (см. `iconography.md`).

**Биотоп для Rich-content (формула листовки):** В джунглях корни висят в воздухе. Кора, цеолит и кокос делают грунт рыхлым, почти невесомым. Мох напоминает о влажности тропиков. В горшке мы воспроизводим эту воздушность через рецептуру под монстеру — отсюда фракция и подбор компонентов грунта N°001.

Грунт для антуриума ЗАЗЕМЛИ — отдельная смесь под него, 1.2 л

**Ключевые характеристики (заполнить минимум):**

| pH | 5,5–6,0 |
| Тип упаковки | коробка |
| Дополнительно | дренаж, корневин 5 г, перчатки, листовка-гайд, дневник на год |
| Производитель | ЗАЗЕМЛИ |

**Аннотация (537 знаков) — Founder Quote 3 tone:**

```
Грунт для антуриума 1.2 л. Антуриуму нужно сидеть в коре и кокосе — это эпифит, не торфолюб. Универсальная земля задушит корни за 2-3 месяца.

Состав: торф нейтральный, кора сосновая, кокос, мох сфагнум, цеолит, биочар. Грубоволокнистая фракция 50% — имитация лесной подстилки. pH 5,5–6,0.

Объём 1.2 л — на горшок 12–13 см. Подходит молодому антуриуму или подросшему черенку. Раз в год.

В коробке — дренаж с запасом, корневин 5 г для поддержки корней, перчатки, листовка-гайд и дневник на год.

80+ источников, сборка в Москве. zazemli.com
```

**Видеообложка:** медальон с **N°003** + дудл Антуриума.

**Биотоп для Rich-content (формула листовки):** В тропиках растёт на деревьях, а не в земле. Кора и кокос дают воздух корням-эпифитам, а мох держит влагу. Привык дышать — не потерпит болота. От этого пропорции грунта N°003: 20% коры + 20% кокоса (максимумы линейки) — лесная подстилка для корней, которые в природе сидят на стволе, а не в земле.

## A/B варианты заголовка

Для каждого примера — два варианта Title: **A** = SEO-плотный (главный ключ + объём + бренд), **B** = USP-первый (рецептура/род в начале, ключ остаётся в первых 30 знаках). **A — дефолт; B — тест с партии 0** (решение 4: A/B запускается уже с пилота, не ждать партии 1).

| Пример | A (SEO-плотный, по умолчанию) | B (USP-первый, для A/B) |
|---|---|---|
| 1 · Монстера 3.5 л · Ozon | Грунт для монстеры под горшок 3.5 л · ЗАЗЕМЛИ | Грунт для монстеры — отдельная смесь под неё, 3.5 л · ЗАЗЕМЛИ |
| 2 · Антуриум 1.2 л · Я.Маркет | Грунт для антуриума ЗАЗЕМЛИ — отдельная смесь под него, 1.2 л | Грунт для антуриума ЗАЗЕМЛИ — кора и кокос, 1.2 л |
| 3 · Замиокулькас 2.2 л · Ozon | Грунт для замиокулькаса под горшок 2.2 л · ЗАЗЕМЛИ | Грунт для замиокулькаса — минеральная смесь под него, 2.2 л · ЗАЗЕМЛИ |

## Правила применения

- **Имя «ЗАЗЕМЛИ» обязательно в Title** — последним на Ozon, вторым на Я.Маркете (после типа товара). См. `brand-voice-profile` v1.0.2 §«Sender naming».
- **N° SKU — в видеообложке (медальон)**, НЕ в названии карточки. См. `sku-system.md` §1 «Где НЕ применяется».
- **Domain в Rich-content и в B5** — `zazemli.com` (откатано с `.space` 2026-05-17, решение A).
- **Caveat-лимит:** 0 для основной карточки (Title + 5 буллетов + аннотация), ≤1 в Rich-content. Правило 11: Caveat = только handwritten-пасхалка.
- **Заполнить ВСЕ атрибуты Я.Маркета** до того, как писать аннотацию. Веса алгоритма: атрибуты > текст.
- **Я.Маркет-категория:** «Дом / Сад / Огород / Грунт и удобрения / Грунт для комнатных растений» (решение 2).
- **Ozon-категория:** аналогично.
- **Корневин 5 г** — везде (Ozon-карточка, Я.Маркет-атрибут, B3, B4, аннотация). Не 10 г (решение 3).
- **A/B Title — с партии 0** (решение 4): вариант A — дефолт, B — параллельная тестовая ветка на пилоте 50-100 боксов. Метрики: CTR в выдаче, конверсия в карточку.
- **Биотопы — финальные формулировки (Волна 4, 2026-05-25):** 1-строки для B1 и для product-description берутся из `product-description.md` §«Биотопы по 7 SKU» (источник: БЗ Продукт/Ресечи/karta-issledovanii-SKU.md). Расширенная форма для Rich-content (60–120 слов) — из БЗ Продукт/Ресечи/{SKU}/biotop.md, тон Founder Quote 3, конкретная сцена, бытовая отсылка, «что это значит для горшка». **Запрещено:** «гемиэпифит-душитель», латынь-декорация, страны списком, научная сухость. **Образец стиля** в примерах ниже (Монстера, Антуриум, Замиокулькас).
- **Тон Founder Quote 3** в B2 и B3: конкретика по растениям («Антуриуму», «Монстере», «Замиокулькасу»), противопоставление («не торф — кора», «не пеностекло — мох»), бытовая лексика («навалить пеностекла», «присыпать корни для поддержки», «дренаж с запасом под разные формфакторы»). Решение D.
- **При публикации — прогон через `pre-publish-checklist.md`** (раздел «Маркетинг/Каналы/pre-publish-checklist»).
- **Первое фото** — open-box раскладка содержимого (см. `marketplace-card-ozon.md` §«Первое фото = главная отличка» + `photography.md` категория 1).
- **Один шаблон на 16 SKU.** Все карточки генерируются из единого `sku` config; редактируем только значения переменных.

## Что НЕ применять

Базовый чек-лист — `anti-patterns.md` §5. Точечно для маркетплейса:

- ❌ CAPS LOCK в Title и буллетах (искл. «ЗАЗЕМЛИ» как sender name).
- ❌ Эмодзи нигде.
- ❌ False urgency: «Только сегодня», «Осталось 3 шт», «Успей купить».
- ❌ Слова «уникальный», «инновационный», «премиальный» (про себя), «премиум-бокс», «премиум-грунт» в Title и B1-B2 (выгорело в категории, см. `audit-bz-copy-library.md` §6).
- ❌ Восклицательные знаки массово (точка по умолчанию).
- ❌ Hyperbolic adjectives: «лучший», «номер 1», «революционный».
- ❌ Манифест-эссе и поэтические метафоры (это не сайт; ритуал и поэтика — IG и `/about`).
- ❌ Декоративная латынь (`ex cultura`, `observatio`). Только род растения в Spectral italic, и то только в Rich-content, не в буллетах.
- ❌ Селебрити-цитаты, «как говорит [имя]».
- ❌ «Магия / секретная формула / чудо-средство».
- ❌ Сравнения «vs HURY», «лучше чем [конкурент]» — премиум не сравнивается.
- ❌ «Натуральный / экологичный / эко-» без сертификата (Я.Маркет может скрыть карточку).
- ❌ «Добавить в корзину», «Оформить заказ», «Заказ отгружен» в любом тексте.
- ❌ «Осознанность» (выжжено категорией wellness-инфлюенсеров; см. канон 2026-05-17).
- ❌ Биотоп от себя — только из `product-description.md` §«Биотопы по 7 SKU» (источник БЗ Продукт/Ресечи). Расширенная форма — из БЗ Продукт/Ресечи/{SKU}/biotop.md.
- ❌ Корневин 10 г, 7 г и т.п. — только 5 г (решение 3).

## Контр-пример (компактный)

**Плохой Title (нарушений: 4):**

```
ПРЕМИУМ грунт для всех растений — уникальная формула! Только сегодня скидка!
```

Нарушения:
1. CAPS LOCK в потоке («ПРЕМИУМ»).
2. «уникальная формула» — blacklist, выгорело.
3. «Только сегодня скидка» — false urgency + восклицание.
4. «для всех растений» — разрушает позиционирование «один SKU = одно растение».

**Плохой B2 (нарушений: 3):**

```
Лучший премиальный грунт для вашей любимой монстеры! Эко-формула для всех ароидных.
```

Нарушения:
1. «Лучший», «премиальный» — blacklist (хайп + про себя).
2. «Эко-формула» — blacklist (без сертификата).
3. «Для всех ароидных» — абстракция, противоречит Founder Quote 3 («не семейство ароидных — для Антуриума отдельно»).

## Чек-лист 10 anti-pattern-ов

Прогнать перед публикацией каждой карточки. Все 10 — ❌ (отсутствуют).

- [ ] Нет CAPS LOCK в Title и буллетах (искл. «ЗАЗЕМЛИ» как sender name)
- [ ] Нет эмодзи
- [ ] Нет false urgency («только сегодня», «осталось N»)
- [ ] Нет слов «уникальный / премиальный / инновационный / премиум-бокс»
- [ ] Нет восклицаний массово (точка по умолчанию)
- [ ] Нет hyperbolic adjectives («лучший / номер 1 / революционный»)
- [ ] Нет манифеста и поэтических метафор в буллетах
- [ ] Нет декоративной латыни (только род растения в Spectral italic в Rich-content)
- [ ] Нет селебрити-цитат
- [ ] Нет «магии / секретной формулы / чудо-средства»
- [ ] **Конкретика по растению (не семейство); корневин 5 г; биотоп — placeholder; домен `.com`**
