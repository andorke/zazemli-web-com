## Что это

Принципы и бриф для фотосессии партии 0. Что снимаем (категории), как снимаем (свет / композиция / цвет), на что ориентируемся (бенчмарки).

## Роль в ДНК-четвёрке

Источник **02 Apothecary** (галерейная строгость, sparse-композиция) + источник **03 Naturalist** (материальность, тактильность, фактура земли). См. `02_voice/dna.md` §1, §7.

## Бенчмарки (master-brief §4.4)

| Бренд | Что берём |
|---|---|
| **Aesop** | Upcoated-фоны, мягкий боковой свет, dramatic chiaroscuro, тактильность материалов. Фон bone / chalk / dark stone. |
| **Diptyque** | Premium-кадрирование, шрифт виден в плоти, «парижская строгость» в композиции. |
| **ММОМА** | Музейная строгость, белый куб как контейнер, объект в центре, ничего лишнего. |
| **Roots / Крапива** (Москва) | Природный свет, растение в естественной среде, без «деко-стилизации». |

**НЕ берём:** Instagram-блогерская эстетика (overexposed, тёплые фильтры, рука-в-кадре), Etsy-кейс (deco-витрина), фуд-фото (top-down с овощами вокруг).

## 4 категории фотосета партии 0

### Категория 1 · Распаковка (для Ozon · первое фото карточки)

**Цель:** показать «что внутри» — главная отличка от HURY (master-brief §11.3 CMO).

**Сцена:** открытая коробка с раскладкой содержимого по уровням (см. `05_production/box-layers.md`):
- Уровень 05: листовка-гайд лицом вверх.
- Уровень 04: чёрный конверт + видимые перчатки + пакет корневина.
- Уровень 03: грунтовый пакет с окошком (виден состав).
- Уровень 02: дренажный пакет.
- Уровень 01: сотовая крафт-бумага (видна).

**Свет:** мягкий боковой, 45° слева. Лёгкие тени для тактильности.

**Фон:** bone-стол (старая столешница / неокрашенная сосна) + нейтральная тень.

**Композиция:** top-down (vertical 4:5 для Ozon).

**SKU-цвет:** проявляется через медальон Ø100мм на видимой стенке коробки.

### Категория 2 · Компоненты грунта (для `/lab` + IG-карусель)

**Цель:** apothecary-каталог — 11 материалов как «образцы вещества».

**Сцена:** 11 круглых блюдец на bone-фоне, в каждом — горка материала (керамзит, торф, цеолит, кокос, мох, биочар, песок, диатомит, пеностекло, кора, биогумус). Лейблы Unbounded uppercase.

**Свет:** flat, без теней.

**Фон:** bone-картон или белая бумага.

**Композиция:** grid 4×3 или 3×4. Идеально top-down с tripod.

### Категория 3 · Ритуал пересадки (для IG-Reels + `/guide`)

**Цель:** 4 шага ритуала визуально (00 ПРИХОД — opt./v2+, 01 ЗАБОТА, 02 ДРЕНАЖ, 03 ГРУНТ).

**Сцена:** руки в кадре, перчатки, действие. ASMR-настрой — медленные движения, фокус на материале.

**Свет:** мягкий, естественный (окно).

**Фон:** деревянный стол + газета.

**Композиция:** vertical 9:16 для Reels.

**4 видео по 15-30 сек** + макроcuts материалов.

### Категория 4 · Финальный кадр (растение + бокс после ритуала)

**Цель:** before/after для Ozon (последнее фото) + конверсия на сайте.

**Сцена:** растение в новом горшке, дневник лежит рядом, пустая коробка на заднем плане.

**Свет:** dramatic — окно слева, бокс в полутени, растение в свете.

**Фон:** интерьер «цифрового человека» — стол с ноутбуком, окно, нейтральный текстиль.

**Композиция:** horizontal 16:9 для Ozon hero / vertical 4:5 для IG.

## Технические требования

| Параметр | Значение |
|---|---|
| Разрешение | Минимум 2400px по длинной стороне (для Ozon zoom + retina-веб). |
| Формат | JPG (sRGB, quality 85+) для веба / TIFF (Adobe RGB) для архива. |
| Цветовой профиль | sRGB для веба, Adobe RGB для печати (конверсия по `print-web-bridge.md`). |
| Ozon hero | 4:5 vertical (1200×1500 минимум). |
| IG feed | 4:5 vertical (1080×1350). |
| IG Reels | 9:16 vertical (1080×1920). |
| Site hero | 16:9 horizontal (1920×1080) — для page-hero opt. |

## Правила применения

- **На первом фотосете** — снимать категорию 1 (распаковка) обязательно. Остальные — по бюджету.
- **Sku-color** на фото проявляется через медальон (Ø100 мм), не через цветной фон.
- **SKU-доступность** — снимать после получения тиража партии 0 (есть медальоны, есть листовки).
- **Перед публикацией** — color correction в sRGB, проверка на 3 устройствах (iPhone, Android, MacBook).

## Что НЕ применять

См. `02_voice/anti-patterns.md`:
- Stock-фото растений.
- Overexposed «солнечное окно».
- Тёплые Instagram-фильтры (VSCO A1, A4, etc).
- Рука-в-кадре без действия (рука с горшком = OK; рука держит коробку «эстетично» = НЕТ).
- Top-down с продуктом + деко-вокруг (Etsy-стиль).
- Цветной фон, кроме bone / chalk / тёмная сосна / нейтральный стол.
- Filter эффекты в постобработке (clarity > 30, vibrance > 20).
