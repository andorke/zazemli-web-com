## Что это

Длинное продуктовое описание для страницы сайта `/collectio/[plant]` (например, `/collectio/monstera-deliciosa`). Жанр работает на стыке витрины и обучения: покупатель уже знает категорию (грунт для пересадки) — здесь мы переводим его из **Solution Aware → Product Aware**, объясняя, почему именно эта рецептура и именно этот бокс. **Это НЕ Ozon-карточка** — другой канал, другой жанр (см. `marketplace-bullet`, Волна 2).

## Роль в ДНК

Доминанты `/collectio/[plant]` — самый сбалансированный носитель бренда: **Галерист 30 · Фармацевт 45 · Натуралист 15 · Друг 10**.

## Правила применения

- Только сайт `/collectio/[plant]` (домен `zazemli.com`). **НЕ Ozon** — там `marketplace-bullet` (Волна 2).
- SKU-цвет — только через `var(--sku)` в MaterialDot, hover CTA, разделителях KickerHeader. В body — `--text-charcoal`. AA-rule: cosmos/iris/buttercup на bone в тексте <24px запрещены (см. `sku_color_aa_rule.md`).
- **Caveat ≤1 на product-page** — в hero ИЛИ в «Ритуале», не в обоих.
- **Биотоп** — формула листовки (3 строки) из §«Биотопы по 7 SKU». Стиль ToV Насты, выверенный для печати. Без «гемиэпифит», без списков стран (упрощённая география), без латыни-декорации.
- **Латынь — только род в нижнем регистре**, Spectral italic, `--text-latin` (moss). Например: `monstera`, не `monstera deliciosa`. Без вида (зафикс 2026-05-25 Настей: «не употребляем сорт в названиях»).
- **N° SKU** — на сайте как маркер коллекции (ApothecaryBar-стилистика), это OK. **НЕ употребляется** на маркетплейсах и в пушах в тексте (только в визуале медальона).
- **HERO** — даём 2 версии для A/B-теста: вариант **A** = характерная фраза из листовки (острее, для retargeting / social-share); вариант **B** = базовый Lvl 2 («Заземли {растение}. Руки в землю — голова свободна.») для главной страницы и SEO-сниппета.
- **РИТУАЛ Caveat ≤1** — даём 2 версии: **A** = handwritten пасхалка из словаря бренда («Корни скажут спасибо»); **B** = характерная фраза листовки курсивом. Тест через 2 недели после релиза.
- В hero вместо «премиум» — «curated», «авторский», «одна пересадка — один бокс».
- Slogan в hero — только Lvl 2. Lvl 3 — главная, Lvl 4 — `/about`.
- **«Рецептура под род» — запрещена** (зафикс 2026-05-25). «Род» — биологический термин, не в обиходе. Использовать: «грунт под монстеру отдельно», «не универсальный», «отдельная смесь под каждое растение».
- **NASA-claims запрещены** для Монстеры (N°001), Антуриума (N°003), Замиокулькаса (N°006) — отсутствуют в NASA Wolverton 1989 (источник: `karta-issledovanii-SKU.md` тема 19, юр.фильтр).
- **Дисклеймер токсичности** на MVP не выводится (решение Насты 2026-05-25, вариант B) — появится в фазе 1.5 как отдельная kb-faq-short.

## Что НЕ применять

Полный список — `anti-patterns.md §5`. Точечно по жанру: селебрити-цитаты, «магия / секретная формула / уникальный», hyperbolic adjectives, восклицания массово, эмодзи и декоративные символы, КАПС в потоке, «эко / органик / 100% натуральный», маркетплейсная лексика («оформить заказ», «в корзину»), «С теплом и любовью, команда».
