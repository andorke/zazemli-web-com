## Правила Spotlight

- **1–2 строки** (40–80 знаков, читается одним взглядом)
- **Деталь биотопа + образ**, без перегиба в поэтику
- **Только род** (без вида и сорта — нет «monstera deliciosa», «ficus benjamina»). Memory `feedback_only_genus_in_copy`
- **Не сравниваем SKU** между собой. Memory `feedback_no_collection_compare`
- **География упрощена:** «тропические джунгли», «тропики ЮВ Азии», «Южная Америка», «Африка», «тропическая лесная подстилка». Без широт, страт, цифр. Memory `feedback_deep_tov_structure`
- **Без §, цифр природы, имён ботаников, годов**
- **Тёплый тон** — это статья куратора, не реклама и не таблица
- **Не повторяет** биотоп-формулу листовки дословно (та — для развёрнутого использования на /lab Phase 1.5)
- **Соединяет «землю» (биотоп)** через «заботу» (правильный грунт) — рациональный proof для эмоциональной core formula

## 7 Spotlight цитат · финал

Каждый SKU — главный вариант **A** + альтернатива **B** (Настя выбирает 1 из 2 в Figma при финале макета).

### N°001 · МОНСТЕРА · *monstera*

**A.** Тропические джунгли. Корни ищут не землю — воздух.

**B.** Из джунглей, где растение тянется наверх и не помнит земли.

---

### N°002 · ФИКУС · *ficus*

**A.** Дерево, помнящее ритм тропиков ЮВ Азии. Привыкает однажды — надолго.

**B.** Тропики ЮВ Азии. Растение, которому важен один дом и один свет.

---

### N°003 · АНТУРИУМ · *anthurium*

**A.** В тропиках живёт на деревьях, а не в земле. Корням важен воздух.

**B.** Из тропиков, где корни обнимают дерево, а не почву.

---

### N°004 · АГЛАОНЕМА · *aglaonema*

**A.** Тропическая лесная подстилка. Тень, влага и рисунок на листе.

**B.** С нижнего яруса тропического леса — где живут тень и кислая земля.

---

### N°005 · СПАТИФИЛЛУМ · *spathiphyllum*

**A.** Тенистый подлесок Южной Америки. Спокойное растение для нешумного места.

**B.** Подлесок Южной Америки. Просит воды листьями, не голосом.

---

### N°006 · ЗАМИОКУЛЬКАС · *zamioculcas*

**A.** Из Африки, где дожди — событие. Запас воды живёт в клубнях.

**B.** Африка. Растение, которое научилось ждать.

---

### N°007 · ЭПИПРЕМНУМ · *epipremnum*

**A.** Лиана влажных джунглей. Цепляется — и идёт дальше.

**B.** Из влажных джунглей, где лиана становится дорогой для других.

│ Тропические джунгли.        │ ← Spotlight · Spectral italic 300 · 14-15px · charcoal-soft

**Цветовая логика SKU-цвета:**
- SerialNumber `N°001` окрашивается в SKU-цвет растения (Монстера = moss, Фикус = rose, и т.д.)
- Spotlight всегда `charcoal-soft`, не SKU-цвет (для читаемости + единства витрины)
- Эпипремнум N°007 = moss (как N°001, по memory `epipremnum_moss_color`)

## Чем Spotlight отличается от других копи-форматов

| Формат | Длина | Где | Голос |
|---|---|---|---|
| **Spotlight** (этот файл) | 1–2 строки, 40–80 знаков | Карточка `/collectio` MVP, под названием | Куратор музейной витрины — образно, тепло, без сравнений |
| Биотоп-формула (листовка) | 3 предложения | Welcome-листовка в боксе + развёрнутые места | Аптекарь — конкретно, с долями компонентов |
| Характерная фраза | 4–7 слов | Welcome-листовка back + IG-карусели + диалог поста | Caveat-handwritten, разговорная пасхалка |
| Sexy-fact (`sexy-facts-7-sku.md`) | 1–3 предложения | IG-карусели, блог Phase 2, Caveat-блоки | Любопытный факт, нейтральный научно-популярный |

**На MVP-витрине Spotlight нужнее всех остальных** — это единственная копи-связка между фото и SerialNumber на статичной карточке без клика.

## Применение в Figma (для финала макета `/collectio`)

1. Каждая из 7 карточек CollectionGrid получает Spotlight (вариант **A** по умолчанию)
2. Если Настя при сборке выбирает **B**, делает замену на одной карточке — не глобально
3. Spotlight не подсвечивается hover'ом (по принципу `mvp_no_animation`)
4. На mobile Spotlight остаётся 2 строки — никакого collapse/truncate
5. Шрифт: `Spectral italic 300` (по DS-контракту v1.2 — 3 шрифта max, mono не используется)
