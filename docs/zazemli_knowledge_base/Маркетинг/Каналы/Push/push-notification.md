# push-notification · Версия: 1.0.1 · Дата: 2026-05-16

Источник: `genre-templates.md §3` + `brand-voice-profile v1.0.1` + мини-ретро Волны 1 (решения Насти Q-5, Q-6, Q-7', Q-8).

Покрывает 5 триггеров для 4 каналов (TG-бот, email-push, web-push, маркетплейс-пуш Ozon/Я.Маркет). Универсальный лимит ≤30/≤80/≤15 — по самому жёсткому каналу.

---

## 1. Что это

Шаблон коротких пушей для 4 каналов с единым лимитом: заголовок ≤30, тело ≤80, CTA ≤15 знаков (с пробелами). Регистр базовый — Подруга, альтернативный — Наставник для триггера «Прибытие на ПВЗ» (нужна предметность: адрес, срок, код).

Push — самый рискованный жанр голоса: легко скатиться в «АКЦИЯ! Только сегодня!». Шаблон ставит рамки, в которых это невозможно.

---

## 2. Имя отправителя (обязательное поле)

**Жёстко — «ЗАЗЕМЛИ» (КАПС, display name по мастер-брифу §7.7) для всех каналов:**

| Канал | Поле | Значение |
|---|---|---|
| TG-бот | имя бота / display name | `ЗАЗЕМЛИ` |
| Email-push (transactional) | from-name | `ЗАЗЕМЛИ` |
| Web-push | title (sender) | `ЗАЗЕМЛИ` |
| Маркетплейс-пуш (Ozon, Я.Маркет) | display sender | `ЗАЗЕМЛИ` |

КАПС применяется ТОЛЬКО к имени отправителя (display name). В теле пуша КАПС запрещён (см. §8).

## 3. JTBD-тройка получателя

- **Functional:** узнать важное событие за секунду, без открытия приложения.
- **Emotional:** почувствовать, что меня помнят (не «забыли отправить», не «ещё одна рассылка»).
- **Social:** иметь бренд, который не спамит — push приходит, потому что есть повод, а не потому что есть KPI.

## 5. Каналы и лимиты

- **Push общий (web/email/TG):** заголовок ≤30 / тело ≤80 / CTA ≤15.
- **Web-push:** то же.
- **TG-бот:** ≤120 знаков всего (заголовок + тело + CTA).
- **Маркетплейс-пуш (Ozon/Я.Маркет):** свой CTA-столбец (см. §6) — переход уже внутри маркетплейса, «Купить на Ozon» здесь = тавтология.

Лимиты — посимвольно, с пробелами. Превышение запрещено даже на 1 знак.

## 8. Регистры по триггерам

- **Подруга** (4 триггера): Новая коллекция · Полив · Ответ в Direct · Выезд бокса.
- **Наставник** (1 триггер): Прибытие на ПВЗ (техническая ясность — адрес, срок, код).

---

## 9. Правила применения

1. **Имя отправителя = «ЗАЗЕМЛИ»** — жёстко, во всех каналах. См. §2.
2. **Caveat в push = 0** — жёстко. Никаких handwritten пасхалок, никаких приписок голосом Насти. Push — слишком короткий и операционный жанр, любой Caveat = шум.
3. **Частота «Полив»-пуша** — НЕ копи-правило. Определяет логика дневника QR. Задача команде продукта (Иван). Копирайтер не нормирует частоту здесь.
4. **Эмодзи = 0** — нигде, ни в каком триггере (бренд-правило, brand-voice-profile §10.2).
5. **КАПС в теле = 0** — только в имени отправителя «ЗАЗЕМЛИ» как display name. В заголовке и теле — нижний регистр с одной заглавной.
6. **Восклицания** — 0 в массе. Опционально 1 в специальных триггерах (новая коллекция), но точка по умолчанию.
7. **CTA — глагол одного слова** или короткое действие. Не «Перейти к ознакомлению».
8. **Один пуш — одно действие.** Не «открой и расскажи в сторис и оставь отзыв».
9. **Контекст темы обязателен** для Direct и Полив (без него пуш = шум).
10. **«Бокс заземлён»** — только для триггера «Выезд». Не повторять в «Прибытие» (будет дешёвая повторюшка).
11. **Маркетплейс-пуш** — использовать столбец «CTA маркетплейс-пуш» из §6, не дублировать «Купить на Ozon» внутри Ozon-канала.
12. **Время отправки:** Новая коллекция — будни 11:00–14:00. Полив — утро 8:00–10:00 (частоту определяет дневник, см. п.3). Выезд/Прибытие — по событию. Direct — по событию, без ночной отправки (после 22:00 — не пушим).

---

## 10. Что НЕ применять

- См. полный список: `anti-patterns.md §5`.
- **Точечно для push:** false urgency («Только сегодня»), FOMO («Осталось 3»), КАПС в теле, эмодзи, длинные слоганы Lvl 3–4, маркетплейсные глаголы («Оформите», «Добавьте в корзину», «Отгружен»), сюсюкание («Привет, дорогой!»), декоративная латынь, подпись «С теплом и любовью», обращения «уважаемый клиент», «земляне», Caveat-приписки.

## 11. Контр-пример (как НЕ писать)

**Триггер: Выезд бокса**

`Уважаемый клиент, ваш заказ №12345 был отгружен. Срок доставки — в кратчайшие сроки.`

Что нарушено:
- «уважаемый клиент» — blacklist (word-list).
- «отгружен» — маркетплейсный глагол (anti-patterns §5.6).
- «в кратчайшие сроки» — канцелярит, blacklist.
- Нет фирменного «бокс заземлён».
- Превышение лимита тела (84 знака против ≤80).

**Как надо:** `Бокс заземлён.` / `Едет к тебе. Трек внутри, новая дата — пятница.` / `Отследить`.
