## Что это

Шаблон сценария Reels-пересадки в формате таймлапса для IG.
Демонстрируем процесс заземления растения: руки, материалы, шаги — ускоренно, эстетично, без голоса (опц. voice-over Подругой 1–2 фразы), под фоновую музыку или тихие естественные звуки записи (без перемикширования и амплификации).

Жанр-сосед — `ig-reflection` (статичный пост-рефлексия, не видео). Не дублирует его: тот про слова, этот про процесс.

## Роль в ДНК + канал обучения

- **Канал:** IG Reels (tone-matrix.md → IG = обучаем 15–60 сек через личное проживание ритуала).
- **ДНК-доли (dna.md §7, ближе к строке «IG Reels»):**
  - Натуралист **50%** — материал в действии, фактура грунта в кадре.
  - Галерист **30%** — композиция кадра, центр, воздух, один объект.
  - Друг **15%** — N°-медальон + опц. voice-over Подругой + рукописный капшн.
  - Фармацевт **5%** — N°-маркер, COLLECTIO-kicker в обложке Reels.
- **Mostly visual identity:** константа (палитра bone/charcoal/moss + SKU-цвет, Unbounded в обложке, N° на медальоне).
- **Format:** подчинён алгоритму IG (9:16, ≤60 сек), но композиция кадра — апотекарь × галерист.

## Структура сценария — базовая (15–30 сек)

| Сек | Кадр | Звук | Текст на экране |
|---|---|---|---|
| 0–3 | **Cold-open:** рука кладёт N°-медальон рядом с пустым горшком на bone-фон. Композиция — галерейная, один объект, воздух | фоновая музыка (тихая, инструментал) **ИЛИ** тихий шорох коры, естественный | (опц.) `N°001 · Монстера` — Unbounded 500, charcoal, op 0.8 |
| 3–18 | **Процесс (таймлапс):** дренаж в горшок → присыпь грунтом → корни в центр → засыпь субстратом → утрамбуй | продолжение муз. дорожки **без склеек** | (опц.) шаги «Дренаж · Грунт · Корни · Субстрат» — Spectral italic, op 0.7, сменяются в такт |
| 18–25 | **Финал:** растение в новом грунте, общий план сверху, ❦ в углу | музыка | **слоган Lvl 0** «Заземли себя.» — одна строка (правило 17: короткие = Lvl 0) |
| 25–30 | **Капшн-промо в кадре:** wordmark ЗАЗЕМЛИ + `zazemli.com` + `@zazemli_collectio` | музыка финал (затухание) | домен + handle, Unbounded 500 |

## Структура сценария — расширенная (30–60 сек, медитативный)

Дополнительно к базовой:

- **0–5 сек** — расширенный cold-open: «приготовления» (перчатки, открытый бокс, материалы разложены).
- **Между шагами процесса** — 1–2 крупных плана материалов (керамзит сыпется, кора в ладони, мох в кадре) — **в реальном времени, не таймлапс**. Это контраст: общий таймлапс ускорен, крупные планы медленные.
- **45–55 сек** — финал с продуктом в горшке + дневник растения рядом + N°-медальон в кадре. **Слоган Lvl 2** «Руки в землю. / Голова свободна.» — две строки (правило 17: длинные = Lvl 2).
- **55–60 сек** — капшн-промо.

Логика: в коротком (15–30 сек) — динамика и польза, Lvl 0. В длинном (30–60 сек) — медитация и эстетика, Lvl 2.

## Звук — правила

| Источник звука | Можно | Нельзя |
|---|---|---|
| **Фоновая музыка** | Тихий инструментал, лицензионная (Artlist, Epidemic Sound, Audio Library с разрешением). Темп спокойный | Trending Sounds IG автоматом, хип-хоп, мемные треки, бренд-джинглы, поп с вокалом |
| **Естественные звуки** (опц.) | Шорох коры, пересыпание грунта, постукивание керамзита по горшку — **естественно**, в записи камеры | ASMR-перемикширование, амплифицированное постукивание, шёпот, перемещённый микрофон вплотную к материалу |
| **Voice-over** (опц.) | 1–2 фразы регистром Подруги — спокойно, ровно, без «театральности». Можно **без слов вообще** | Шёпот, придыхание, нарочитая «медитативная» интонация, гайд-голос Наставника, манифест-голос Редактора |
| **Тишина** | Возможна, если музыка задаёт настроение | Полная тишина без саундтрека = «брошенное видео» |

Принцип: **звук обслуживает кадр, не наоборот.** Если звук перетягивает внимание — переделай.

## Правила применения

- **N°-медальон в кадре** — обязательно в cold-open и финале. Каноничный приём бренда (sku-system.md §1 + iconography.md «1-ring · moss»).
- **Палитра кадра:** bone-фон, charcoal-перчатки/инструменты, moss-медальон или SKU-цвет растения (см. sku-system.md §3 — terracotta для N°003, ochre для N°006 и т.д.).
- **Композиция кадра:** галерейная (один объект в центре, воздух 40–60% площади).
- **Лексика капшна** — whitelist (word-list.md): «заземли», «бокс», «грунт», «корни», «ритуал», «рецептура», «коллекция», «руки в землю», «голова свободна». **Конкретика по растению, не по семейству** (Founder Quote 3): «Монстере — мох», «Антуриуму — кора», «Замиокулькасу — пеностекло».
- **Domain в капшне:** `zazemli.com` (откатано с `.space` 2026-05-17, решение A).
- **Handle в капшне:** `@zazemli_collectio`.
- **Caveat в капшне:** ≤1 (IG общее ≤2, но здесь точечно ≤1 — формат короткий, второй Caveat = декор). Правило 11: Caveat = handwritten-пасхалка, не свободная авторская нотка.
- **Эмодзи в капшне:** 0 (blacklist, voice-principles.md TOV §1.3).
- **Музыка** лицензионная, не Trending Sounds.
- **Финальный CTA капшна** спокойный: «Бокс — на нашей странице», «zazemli.com», «по ссылке в шапке». Не «Купи сейчас!», не «Скидка!».
- **Slogan-правило 17 (решение Насты):**
  - **15-25 сек Reels (быстрый таймлапс)** → слоган **Lvl 0** «Заземли себя.» — одна строка.
  - **30-60 сек Reels (медитативный)** → слоган **Lvl 2** «Руки в землю. / Голова свободна.» — две строки.
  - **Не наоборот** — Lvl 2 в коротком ролике не помещается визуально, Lvl 0 в длинном выглядит обрубленным.
  - **Lvl 3 и Lvl 4** не используем — это сайт и /about.

| Запрет | Почему |
|---|---|
| ASMR-шёпот, нарочитые амплифицированные постукивания, перемикшированные звуки коры | Решение Насты на мини-ретро Волны 1 (Q-5 = 1) — «слишком крипово», ломает регистр Подруги, тянет в эстетику, чуждую бренду |

## Контр-пример (компактный)

**Плохо — `N°005 Спатифиллум 2.2 л` · 35 сек:**

> Кадр 0–3: лицо Насти крупным планом, шёпотом «сегодня заземляем спатифиллум...».
> 3–25: процесс под Trending Sound (хип-хоп, 130 BPM), ASMR-постукивание керамзита, амплифицированное.
> 25–35: финал, капшн «N°005 СПАТИФИЛЛУМ! 100% натуральный грунт! Только сегодня скидка 20% — приобретайте по ссылке! #эко #органик». На экране — Lvl 0 «Заземли себя.» (для 35-сек ролика должно быть Lvl 2).

**Что нарушено:**
1. Лицо Насти крупно (она = бренд, не герой) + шёпот = ASMR (отдельно запрещено решением Насты).
2. Trending Sound + хип-хоп + амплифицированные постукивания = не бренд-эстетика, не апотекарь.
3. Капшн: КАПС в потоке, blacklist-слова («100% натуральный», «эко», «органик»), маркетплейсный CTA («приобретайте», «скидка»), `#эко` и `#органик` — юридически невалидно.
4. **Lvl 0 слогана в 35-сек ролике** — нарушение решения 17 (должно быть Lvl 2).
5. **Абстракция в капшне** — нет конкретики по растению (нарушение Founder Quote 3).

**Сколько правил сломано:** 8 из 14 «нельзя». Это пересъёмка целиком.

## Чек-лист 11 anti-pattern-ов (применять перед публикацией)

1. [ ] Нет ASMR-шёпота, амплифицированных постукиваний, перемикшированных звуков.
2. [ ] Музыка лицензионная, не Trending Sounds.
3. [ ] Лицо Насти не крупным планом во время процесса (опц. — короткий кадр рук, спины, не лица).
4. [ ] N°-медальон в cold-open и финале.
5. [ ] Палитра кадра — bone / charcoal / moss или SKU-цвет.
6. [ ] Капшн: ≤120 знаков с handle, без эмодзи, без КАПС в потоке, без blacklist-слов, без `#эко` и подобных.
7. [ ] Caveat в капшне ≤1 (handwritten-пасхалка, не декор; правило 11 brand-voice-profile v1.0.2).
8. [ ] Domain `zazemli.com`, handle `@zazemli_collectio`.
9. [ ] **Slogan-правило 17:** для роликов 15-25 сек — Lvl 0; для 30-60 сек — Lvl 2. Не наоборот.
10. [ ] CTA спокойный, без «АКЦИЯ!», «Не пропусти!», «приобретайте».
11. [ ] **Конкретика в капшне** по растению (Founder Quote 3): «Монстере — мох», «Антуриуму — кора», не «грунт под род».

Дополнительно — общий чек-лист 10 anti-pattern-ов копирайтера (`brand-voice-profile.md` v1.0.2 §10 anti-pattern-ов).
