## Что это

Шаблон-набор для платной рекламы в 3 каналах: **Ozon-промо** (CPO, трафареты), **Яндекс.Директ** (поиск + РСЯ), **VK/IG-промо** (таргет в ленте и Stories). Один артефакт — три формата, потому что регистр и плотность переключаются по каналу, но whitelist лексики и слоган-матрёшка едины.

Не путать с `marketplace-bullet.md` — это карточка Ozon (органика на странице товара). А `ad-copy` — реклама поверх карточки, трафарет, выдача Яндекса, лента VK/IG.

## Роль в ДНК + канал обучения

Платная реклама — самый «маркетплейс-агрессивный» жанр в воронке. Поэтому здесь мы **переводим**, а не обучаем. Обучение начинается дальше — в карточке Ozon, на сайте, в IG-органике.

- **3 канала × тон обучения**: Ozon-промо — НЕ обучаем (3-5 сек на первый клик); Я.Директ — НЕ обучаем (1-2 сек на скан); VK/IG-промо — обучаем мягко через крючок (15-30 сек на «зачем это»).
- **Доминанты ДНК зависят от канала** (см. `dna.md` §7):
  - **Ozon / Я.Директ** = Фармацевт 60% (точные дозы, N°, объём, цифры) + Натуралист 25% (растение, латынь, рецептура) + Галерист 10% + Друг 5%.
  - **VK/IG-промо** = Друг 50% (голос Подруги, ритуал, проживание) + Натуралист 30% (растение в центре) + Галерист 15% + Фармацевт 5%.

## JTBD / Schwartz / Регистр (под канал)

| Канал | JTBD-фокус | Schwartz | Регистр | Лимиты (знаки) |
|---|---|---|---|---|
| Ozon-промо | Functional «найди под моё растение и горшок» | Most Aware (retargeting / запросы по растению) | Наставник | заголовок ≤35 + 2 буллета по ≤60 |
| Я.Директ | Functional «грунт для конкретного растения» | Solution Aware (поиск) | Наставник | заголовок ≤56 + текст ≤81 |
| VK/IG-промо | Emotional + Social «забота через действие, выглядит хорошо» | Problem Aware (лента, серфинг) | Подруга | заголовок ≤80 + текст ≤220 + CTA ≤15 |

Schwartz — по 5-уровневой шкале awareness Юджина Шварца (Unaware → Problem Aware → Solution Aware → Product Aware → Most Aware). На Ozon-промо обычно ретаргет уже знающего покупателя — Most Aware. В Я.Директе человек ищет решение — Solution Aware. В VK/IG он сидит в ленте без запроса — Problem Aware.

## Состав / структура текста

### Ozon-промо

```
[Заголовок ≤35] → формула «Грунт для [растения] · [объём]»
[Буллет 1 ≤60] → польза-рецептура: «11 компонентов под её корни — …»
[Буллет 2 ≤60] → отстройка через конкретику Founder Quote 3 («не торф — кора и кокос», «не семейство ароидных — Антуриум отдельно»)
```

- N° SKU **не** в заголовок (см. `sku-system.md` §1 «Где НЕ применяется»).
- Имя «ЗАЗЕМЛИ» в заголовок ставится в Я.Директе, на Ozon-промо — не обязательно (там и так под брендом).
- Цены в тексте Ozon-промо **нет** (Ozon сам рисует цену в карточке).

### Яндекс.Директ

```
[Заголовок ≤56] → формула «Грунт для [растения] под горшок [см] — ЗАЗЕМЛИ»
[Текст ≤81] → функциональное обещание + (опц.) цена-якорь «От 1 890 ₽» + CTA «Подробнее»
```

- Имя бренда обязательно в title.
- CTA внутри текста, не отдельной строкой.
- Domain в карточке выдачи — `zazemli.com`.
- Цена-якорь «От 1 890 ₽» допустима **только здесь** — Solution Aware на поиске реагирует на цену.

### VK/IG-промо

```
[Заголовок ≤80] → крючок: переворот ожидания или ритуальный кадр (регистр Подруги). Один из 3 паттернов:
  – «{Растение} — {что ему нужно}. Не {что обычно дают}.» (Founder Quote 3 pattern)
  – «Вот чего вы лишаете {растение}, когда сажаете в универсальный.» (опеннинг Насты дословно)
  – ритуальная сцена («Полчаса. Руки в землю. И ничего больше.»)
[Текст ≤220] → 1-2 конкретики «не X — Y» + ритуал/ощущение + домен
[CTA ≤15] → спокойный глагол: «Заземлить» / «Открыть» / «Посмотреть»
```

- Domain `zazemli.com` — в тексте или в кнопке-ссылке.
- Цены в VK/IG-промо **нет** — здесь работает Problem Aware-крючок и эмоциональный финал, цена сбивает регистр.
- Caveat-приписка ≤1 на всё объявление (правило веба, см. `anti-patterns.md` §2.2 + правило памяти `caveat_print_limit.md`).
- Голос — Подруги: одна разговорная фраза, никакого «приобретите».
- **Конкретика по растениям, не по семействам** — это главное (Founder Quote 3, §«Структурные приёмы»).

## Шаблон + примеры (3 канала × 3 SKU)

### Пример 1 · Ozon-промо · N°001 Монстера 3.5 л

```
Заголовок (26): Грунт для монстеры · 3.5 л
Буллет 1 (56): 11 компонентов под её корни — кора, кокос, мох, цеолит
Буллет 2 (57): Монстере — мох, чтобы держать влагу. Не пеностекло.
```

Разбор: заголовок берёт SEO-формулу `sku-system.md` §4 + объём как «дозу ритуала». Буллет 1 — научный moat бренда (11 компонентов, конкретные имена). Буллет 2 — конкретика из Founder Quote 3 («Монстере, в отличие от Замиокулькаса, нужно удержать влагу — поэтому мох, а не пеностекло»). Никаких эмодзи, восклицаний, КАПСа. **«Не универсальный» не используется** — заменено на конкретное противопоставление.

### Пример 2 · Яндекс.Директ · N°003 Антуриум 1.2 л

```
Заголовок (47): Грунт для антуриума под горшок 12 см — ЗАЗЕМЛИ
Текст (78): Антуриуму — кора и кокос. Не торф. От 1 890 ₽. Подробнее.
```

Разбор: title собран по формуле «ключ + бренд», Solution Aware-запрос «грунт для антуриума» зашит дословно. Текст — конкретика Founder Quote 3 («Антуриуму нужно сидеть в коре и кокосе»), отстройка «не торф» вместо абстрактного «не универсальный», цена-якорь, спокойный CTA «Подробнее». Тире длинное `—`.

### Пример 3 · VK/IG-промо · N°002 Фикус 3.5 л

```
Заголовок (78): Вот чего вы лишаете фикус, когда сажаете в универсальный грунт.
Текст (175): 11 компонентов под фикус — не как для «всех растений». Под горшок 18-20 см. Внутри коробки — дренаж, корневин, перчатки и дневник на год. Руки в землю — голова свободна. zazemli.com
CTA (9): Заземлить
```

Разбор: заголовок — опеннинг Founder Quote 3 дословно («Вот чего вы лишаете растения, когда сажаете в универсальный»). Текст — конкретика «не как для всех растений» (упрощённый паттерн «не X — Y»), три блока: размер, состав бокса, метафора из whitelist. Домен `.com`. CTA — главный глагол бренда. Цены нет (Problem Aware).

## Правила применения

- **Имя «ЗАЗЕМЛИ»** — обязательно в title Я.Директ. На Ozon-промо и VK/IG — опционально (на VK/IG имя в шапке профиля).
- **Domain** — `zazemli.com` везде, где есть ссылка. Короткие ссылки через бренд-сокращатель (если есть), иначе полный домен.
- **Caveat-приписка** — 0 для Ozon-промо и Я.Директ (формат не допускает), ≤1 для VK/IG-промо (правило веба). Правило 11: Caveat = только handwritten-пасхалка из словаря.
- **N° SKU** — не в title и не в тексте объявления. N° живёт на физических артефактах (медальон, дневник, листовка) и в брендинге, не в performance-канале (см. `sku-system.md` §1).
- **Перед публикацией** — прогон через `pre-publish-checklist.md`: лимиты, blacklist, эмодзи/КАПС, домен, регистр под канал.
- **Запреты** — см. `anti-patterns.md` §5.1-5.6 + `word-list.md` blacklist. Особенно для платной рекламы: «премиум», «инновационный», «уникальный», «эко», «органик», «осознанность», «оформите заказ», КАПС в потоке, эмодзи, false urgency, hyperbolic adjectives.
- **Регистр не путать** — Я.Директ и Ozon-промо это Наставник (плотно, по делу), VK/IG это Подруга (живо, разговорно). Подруга в title Я.Директа = промах. Наставник в VK/IG = промах.
- **CTA в VK/IG** — одно слово или 2-3 слова, глагол. «Заземлить», «Открыть», «Посмотреть». Не «Купить прямо сейчас», не «Жми скорее».
- **Объём горшка**, не литраж пакета — это критично для SEO-нейминга (`sku-system.md` §4: отстройка от HURY).
- **Цена** «От 1 890 ₽» — **только** в Я.Директе. В Ozon-промо и VK/IG — без цены.
- **Конкретика растений, не семейств** — «Антуриуму», «Монстере», «Замиокулькасу» (Founder Quote 3 §«Структурные приёмы»). Не «ароидным», не «суккулентам», не «растениям».
- **Лимит «не универсальный»** — максимум 2 повторения на 1 объявление. Дальше — заменять на конкретику «не X — Y» из Founder Quote 3.

## Что НЕ применять

См. `anti-patterns.md` §5 целиком. Точечно для платной рекламы:

- КАПС в потоке («ЛУЧШИЙ ГРУНТ ДЛЯ ВАШЕЙ МОНСТЕРЫ!!!»)
- Эмодзи и декоративные символы Unicode (♡, ★, ✿, ✓)
- False urgency («Только сегодня!», «Осталось 3 коробки!», countdown-таймер)
- Восклицательные знаки в массе («АКЦИЯ!», «СКИДКА!»)
- Слова blacklist: «премиум», «премиум-бокс», «осознанность», «эко-», «органик», «натуральный 100%», «инновационный», «уникальный», «приобретите», «оформите заказ»
- Hyperbolic adjectives («лучший», «единственный», «номер один на рынке»)
- Кричащие CTA («Жми сейчас», «Не упусти», «Купи прямо сейчас»)
- Сюсюкание в VK/IG («Спасибо, что выбрали нас», «С теплом», «Береги себя»)
- Декоративная латынь («ex cultura», «observatio»). Только род растения в формате `monstera`.
- N° в заголовок объявления (это шум для performance-канала, см. `sku-system.md` §1).
- «Грунт под род» / «грунт для ароидных» — абстракция, blacklist по Founder Quote 3.
- Цена в Ozon-промо и VK/IG (только в Я.Директ).

## Контр-пример (компактный)

**Плохо — VK/IG-промо · N°001 Монстера 3.5 л:**

```
Заголовок: ПРЕМИУМ-БОКС ДЛЯ ВАШЕЙ МОНСТЕРЫ! Уникальный грунт для ароидных!
Текст: Эко-органический инновационный грунт для осознанного ухода
       за вашим растением. Только сегодня скидка 30%! Не упустите!
       Закажите прямо сейчас и получите подарок! От 1 890 ₽.
CTA: КУПИТЬ СЕЙЧАС!!!
```

**Что нарушено (4 нарушения):**

1. **КАПС в потоке + восклицания в массе + слова blacklist** («ПРЕМИУМ», «УНИКАЛЬНЫЙ», «ЭКО-», «ОРГАНИЧЕСКИЙ», «ИННОВАЦИОННЫЙ», «осознанного»).
2. **«Для ароидных» — абстракция, не конкретика** (нарушение Founder Quote 3 — должно быть «для монстеры» с противопоставлением «не как для замиокулькаса»).
3. **False urgency + сюсюкающий CTA** («Только сегодня скидка 30%», «Не упустите!», «КУПИТЬ СЕЙЧАС!!!»).
4. **Цена в VK/IG** — нарушение решения 7 (цена только в Я.Директ).

**Как должно быть** (вариант A из примера 3 выше, адаптированный под монстеру):

```
Заголовок: Вот чего вы лишаете монстеру, когда сажаете в универсальный грунт.
Текст: Монстере — мох, чтобы держать влагу. Не пеностекло, как Замиокулькасу. Внутри коробки — 11 компонентов, дренаж, корневин, перчатки и дневник на год. zazemli.com
CTA: Заземлить
```

|---|---|---|
| 1 | Нет эмодзи и декоративных символов | да / нет |

| 3 | Нет восклицаний в массе (≤1 «!» на объявление, в идеале 0) | да / нет |
| 4 | Нет false urgency («только сегодня», countdown, «осталось N») | да / нет |

| 8 | Регистр соответствует каналу (Я.Директ/Ozon = Наставник, VK/IG = Подруга) | да / нет |
| 9 | Лимиты знаков соблюдены (см. таблицу JTBD) | да / нет |
| 10 | Длинное тире `—`, ёлочки `« »`, никаких `"` и `--` | да / нет |
