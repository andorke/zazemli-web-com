### 2. Дизайн · сквозные правила

- [ ] **Палитра:** Bone / Charcoal / Moss + SKU-цвета — соответствуют `color-system.md`
- [ ] **Эпипремнум N°007 = moss** (не свой 7-й цвет) — memory `epipremnum_moss_color`
- [ ] **3 шрифта max:** Unbounded + Spectral + Caveat. Никаких IBM Plex Mono / Courier Prime (решение 27 мая, таск #10)
- [ ] **Sepia палитра НЕ используется** — только Bone (решение 27 мая)
- [ ] **AA-rule:** cosmos/iris/buttercup не используются для текста <24px на bone (memory `sku_color_aa_rule`)
- [ ] **Caveat-лимит:** ≤ 2 на каждую веб-страницу (memory `caveat_print_limit`)
- [ ] **Fleurons всегда moss** — никогда SKU-цвет (memory `fleurons_always_moss`)
- [ ] **Дудлы пропорциональны** — никаких растяжений по одной оси (memory `feedback_doodles_proportional`)
- [ ] **Без анимации** — только статика + CSS hover (memory `mvp_no_animation`)
- [ ] **Typography-first** — фото 40-60% ширины, не full-bleed, не 100vh (memory `typography_first_principle`)
- [ ] **Header не sticky** — часть страницы, скроллится с контентом
- [ ] **Логотип везде**: SiteHeader (Aesop chrome) + SiteFooter (Frama)
- [ ] **5 QR в SiteFooter:** lab, guide, diary-signup, collectio, IG — все ведут на боевые страницы

## Что считается блокером (релиз НЕ выходит)

- Любая из 5 страниц не открывается в Safari iOS или Chrome Android
- Lighthouse Mobile Performance < 70 (порог жёстче цели — 80, но < 70 = блокер)
- Lighthouse Accessibility < 85
- Email-форма не принимает email или не отправляет welcome-письмо
- Ozon-CTA не работает или ведёт не на тот URL
- Yandex.Metrica counter не подключён (≥ 0 точек собирает)
- Нет cookie-баннера + согласия на ПДн (152-ФЗ нарушение)

**Что НЕ блокер (можно отложить на T+1…T+7):**
- Lighthouse Performance 70-79 (между порогом и целью — выйти можно, исправить за неделю)
- 1 из 6 браузеров показывает мелкие визуальные баги (не разваливание)
- 1 из 7 точек метрики не собирает (но 6 работают)
- Двойные пробелы / опечатки в копи (исправить hot-fix'ом)
