## 0. TL;DR · одна страница

**Проблема:** брендбук v3 — двухскоростной. Печатно-апотекарная часть (§01 ДНК, §02 слоган-матрёшка, §03.5 Apothecary, §04 палитра, §05 типографика, §07 Editorial Whisper, §15 ритуал, §18 правила) — **шедевр.** Веб-применения (§11 UI-компоненты, §13 мокапы страниц) — **слабое.** Текущие веб-мокапы говорят на чужом языке (ecommerce-маркетплейс) и нарушают собственные правила бренда (правило 60/30/10 палитры, TOV-запреты).

**Решение:** дизайн-система = **перенос печатно-апотекарной системы в веб** + добавление веб-специфичных слоёв (semantic-токены, atomic-структура, motion-правила). Эстетический язык наследуется у Aesop / Diptyque / MMOMA — они **уже зафиксированы** как референсы (Master Brief §4.4, TOV §1.1), и сходятся в одном принципе: «**типографика как brand asset, материал как фон, спокойная навигация, минимум акцентов**».

**8 якорей DS** (из этапа 1, переносятся в веб):
1. ДНК-четвёрка одновременно (галерейная строгость + минимализм-fashion + материалы грунта + рукописная наивность)
2. Слоган-матрёшка (Lvl 3 на главном экране — Настя уже выбрала)
3. Apothecary 4 якоря (COLLECTIO / N° / 2-frame / apothecary-bar)
4. Правило палитры 60/30/10 (главное игнорируемое правило)
5. Иерархия Unbounded-весов 700/800-900/900
6. Editorial Whisper mode (паттерн hero как у Aesop)
7. Ритуал 00/01/02/03 (паттерн для /guide)
8. §18 токены (opacity-шкала, letter-spacing, frame-веса, fleuron)

**Что добавляем сверху** (этап 3 — теория):
- 8pt spacing grid (запрет произвольных 36/56/72 — только из шкалы)
- Modular type scale ×1.25 (Major Third — editorial spacing)
- Semantic tokens поверх primitive (12–15 семантических ролей)
- Atomic design (atoms / molecules / organisms / templates)
- Motion: opacity + 5–10px translateY, 200–300ms, Material easing
- **Острые углы** (radius 0–2px max — критический отказ от mainstream)
- **Запрет shadows** (тонкие линии вместо лифтов)

**Главное правило, которому подчиняется всё:** «**Каждая страница — экспонат в музее.** Лишнее не входит. Тишина дороже громкости. Типографика говорит, цвет молчит (кроме momentaries SKU-акцентов на странице конкретного продукта).»

## 1. Этап 1 · Изучение базы знаний

### 1.1 Брендбук v3 — двухскоростной

| Уровень | Секции | Качество | Что делаем |
|---|---|---|---|
| **Печатно-апотекарная система** | §01 ДНК · §02 Слоган · §03.5 Apothecary · §04 Палитра · §05 Типографика · §07 Editorial · §08 Дудлы · §15 Ритуал · §18 Правила | Шедевр | Источник правды для DS |
| **Веб-применения** | §11 UI-компоненты · §13 Мокапы | Слабое | Анти-эталон: чему НЕ следовать |

### 1.2 8 якорей из печатной части — для веба

#### 1.2.1 ДНК-четвёрка одновременно (§01)

Брендбук фиксирует 4 стилевых источника, которые должны жить **на одной странице** одновременно:

1. **Галерейная строгость** (MMOMA-vibe) — пустота, воздух, один объект на постаменте
2. **Минимализм + мода** (Aesop-vibe) — fashion-типографика, крупные буквы, уверенный ритм
3. **Материалы грунта** — коричневые, охра, пемза, песок (земляная палитра — фон)
4. **Рукописная наивность** (Caveat) — дудлы, стрелочки, почерк

3 оси контраста: Премиум × Наивное · Цифровое × Материальное · Интеллект × Ритуал.

**Применение в DS:** каждая страница должна показывать **минимум 3 из 4** источников. Это — критерий проверки на этапе 12 (консистентность).

#### 1.2.2 Слоган-матрёшка (§02) — где какой уровень

Брендбук уже зафиксировал распределение по артефактам:

| Lvl | Слоган | Где живёт в вебе |
|---|---|---|
| 0 · 2 слова | «Заземли себя.» | Favicon-подпись, email footer, fallback meta-description |
| 1 · brand-phrase | «Ритуал пересадки.» | Подзаголовок категории на `/collection` |
| 2 · приглашение | «Руки в землю. / Голова свободна.» | Hero сайта (альтернатива Lvl 3) |
| 3 · полная формула | «Ритуал пересадки: заземли растение — заземли себя.» | Главный экран сайта · эпиграф `/about` |
| 4 · манифест 4 строки | (полный) | Тело `/about` |

**H1 главной = Настин выбор «Заземли растение, заземли себя.»** = вариант Lvl 3 без префикса. Полностью в системе.

#### 1.2.3 Apothecary 4 якоря (§03.5) — готовые веб-компоненты

| Якорь | Печать | Веб-применение |
|---|---|---|
| **COLLECTIO-шапка** | Unbounded 500 · 11pt · letter-spacing 1.6–2 · разделитель 0.5–1px op 0.4 · Spectral italic «N° 001 · партия 0» | Kicker-блок над каждой H1 — `KickerHeader` атом DS. Например: «COLLECTIO ZAZEMLI · N° 001 · monstera» над H1 `/collection/monstera` |
| **Серийный N°[партия][SKU]** | Spectral italic · 36pt · SKU-color | Декор на `/collection/[plant]` — отдельный атом `SerialNumber` |
| **2-frame рамка** | 1px op 0.5 + 0.5px op 0.3 + 6px отступ | Опциональный wrapper компонента `Card` — атом `Frame2x` |
| **Apothecary-bar** | `hairline 0.75 op 0.3 · dot · ❦ moss · dot · hairline 0.75 op 0.3` | Готовый `<Divider />` компонент между секциями страниц |

#### 1.2.4 Правило палитры 60 / 30 / 10 (§04) — главное игнорируемое

| Канал | Что | % | Где живёт |
|---|---|---|---|
| 60% | **Bone** `#F6F4F0` | Фон страниц, карточек, секций | Везде по умолчанию |
| 30% | **Charcoal** `#1C1C1C` | Текст, dark-секции | Текст 80%-opacity (титулы), dark-блоки на 1–2 на страницу |
| 10% | **Moss** `#4A7C59` | Signature accent | Fleuron, latin (Spectral italic), accent-кнопки, hover-state |
| **0% по умолчанию** | SKU-цвета (cosmos / iris / buttercup / sky / poppy) | Только когда виден конкретный продукт | На `/collection/monstera` — moss; на `/collection/anturium` — poppy; и т.д. |

**Правило-следствие:** **на главной, /about, /lab, /guide, /dnevnik — НЕТ SKU-цветов кроме moss.** Никаких 6-цветных стрип, никаких разноцветных KPI-блоков. Это противоположность текущим веб-мокапам §13.

#### 1.2.5 Иерархия Unbounded-весов (§05)

| Роль | Вес | Размер (новая шкала Major Third) | Где |
|---|---|---|---|
| Wordmark / display | Unbounded 900 | `clamp(3rem, 8vw, 4.75rem)` (76px max desktop) | Логотип, hero растения |
| Compressed-hero | Unbounded 800–900 | 60–48px | Цифры ритуала 00/01/02/03 |
| Body-hero | Unbounded 700 | 38–30px | H1 страниц, кикеры |
| H2 | Unbounded 500 | 24–20px | Подзаголовки |
| Body | Unbounded 400 | 16px | UI-текст, кнопки, навигация |
| Caption | Unbounded 400 | 12–14px | Метки, подписи, kicker'ы |

#### 1.2.6 Editorial Whisper mode (§07) — паттерн hero как у Aesop

Wordmark / название в центре + **4 whisper-фразы вокруг** (по 2 слева/справа, разноцветные, разные углы) + 2–4 фоновых дудла. Используется для:

- Альтернативная версия hero `/` (вместо вертикального hero — whisper)
- `/about` — манифест-блок
- `/lab` — hero про 11 компонентов

Это **прямой аналог hero Aesop**.

#### 1.2.7 Ритуал 00 / 01 / 02 / 03 (§15) — паттерн для `/guide`

4 шага. Каждый — карточка с:
- Unbounded 900 · 60px цифра (00 — moss-фон, 01–03 — charcoal-фон, цифры moss-цвет)
- Unbounded 900 · 16px название (заглавными)
- Spectral italic · 13px описание
- Caveat-стрелка с подсказкой

Готовый паттерн для `/guide` (заменяет «6–9 шагов с фото» из брифа СТО §5.5 на 4 ритуальных шага). Этой логике 4-шагов также подчиняется блок «что в боксе» на `/collection/[plant]`.

#### 1.2.8 §18 токены — переносятся 1:1

- **Opacity-шкала** (6 основных + 3 спец): 0.8 H1 / 0.75 latin / 0.7 narrative / 0.5 secondary / 0.4 captions / 0.3 dividers + 0.85 Caveat / 0.55 day-labels / 0.06–0.2 watermark
- **Letter-spacing**: 1.6 COLLECTIO kicker / −1 hero / 0 body / 2.5 caption-uppercase
- **Frame-веса**: 3-frame (журнал) / 2-frame (band, листовка, карточка) / 1-ring (медальон)
- **Fleuron ❦**: всегда moss, opacity 0.8, Spectral · 14–88pt

### 1.3 Конкретные ошибки в текущих веб-мокапах (§11 + §13)

| # | Ошибка | Правило, которое нарушено |
|---|---|---|
| 1 | Кнопки «в корзину», «корзина · 0», бейджи «хит / new / last chance» | TOV v1.2 §3 blacklist + бизнес-модель (нет корзины) |
| 2 | Hero чёрный с ЗАЗЕМЛИ caps 56px + 6-цветная SKU-стрипа | Правило 60/30/10: SKU не на главной |
| 3 | Карточки с буквами-плейсхолдерами M/A/A/С в цветной палитре + 2px цветной полосой | Эстетика дешёвого студенческого ricepaper |
| 4 | KPI-блок «11 / 1 / 15 / ∞ кайф» разноцветными цифрами | Правило 60/30/10 + анти-эстетика музейности |
| 5 | Caveat-фраза с сердечком ♡ | TOV v1.3 §1.3 (запрет эмодзи) — нарушение собственного правила |
| 6 | Цветной баннер «СПРАВОЧНИК ГРУНТОВ» на moss-фоне | Эстетика эко-маркета, не музей |
| 7 | Произвольные размеры шрифта 56/32/80/22/18/28 | Нет modular scale — из этапа 3 |
| 8 | Скруглённые углы по умолчанию | DS требует острые 0–2px max |
| 9 | Drop-shadows на product-card | DS запрещает shadows — тонкие линии вместо лифтов |

### 2.1 Карта 5 эстетических референсов

#### 2.1.1 Aesop — главный эстетический ориентир

**Откуда:** [Aesop case study (work.co)](https://work.co/clients/aesop/) + Master Brief §4.4 + TOV v1.2 §1.1.

- **Elegant motion exposing products from navigation** → micro-anim 200–300ms на hover карточек коллекции, не «оживление» страницы.
- **The Aesop Library** как литературный hub → задел на v2 `/journal` (BACKLOG.md уже содержит RECEPTY_SUMMARY one-pager как первый материал).

**Что не берём:**
- Bespoke typeface (Unbounded/Spectral/Caveat уже фиксированы)

#### 2.1.2 Diptyque — серийность и heritage

**Откуда:** [Diptyqueparis.com / candles collections](https://www.diptyqueparis.com/en_us/c/candles-and-home-collections.html) + Master Brief §4.4 + TOV v1.2 §1.1 («минимализм + чувственность»).

- **Размерная сетка S/M/XL** → у нас 1.2/2.2/3.5 л = S/M/L. Структурно совпадает.

**Что не берём:**
- Engraving / персонализация (не наша история)
- Сезонные мега-кампании (наш масштаб другой)

#### 2.1.3 MMOMA — музейная подача

**Откуда:** [mmoma.ru](https://mmoma.ru/) + Master Brief §4.4.

**Что берём:**
- **Типографика как первичная навигация** (не иконки, не «дружелюбные кнопки») → у нас навигация на Unbounded 400 без иконок, без caps-lock.
- **Подача коллекции через количество + классификацию** («6 локаций / 10 000+ работ / direction: avant-garde / naive / contemporary») → у нас на `/collection`: «7 растений · 16 рецептур · 80+ источников» — этот блок уже есть в брифе СТО §5.2.
- **«Один объект на постаменте»** → §01 ДНК-источник 01 «галерейная строгость · White Cube». Применение: на `/collection/[plant]` главное фото-блок — крупно, без украшений.

**Что не берём:**
- Институциональный калейдоскоп выставок (у нас 7 ровно)

#### 2.1.4 Krapiva — материал как brand asset

**Откуда:** [krapivakrapiva.ru / about-us](https://www.krapivakrapiva.ru/page/about-us).

**Что берём:**
- **Материальная база как brand asset** — у них «серый фактурный бетон», у нас earth-textures (`--soil` `--ceramsite` `--pumice` `--sand` `--gravel` `--chalk` `--graphite` из §04). **Это критический insight для DS:** текущие веб-мокапы используют только `bone` как фон. Земляные тона из §04 не задействованы. DS должен поднять земляную палитру как **фоновую подложку** для веб-секций.
- **Эволюция от plant-shop к lifestyle** — у Крапивы добавлены candles+fragrances+cosmetics. У нас Master Brief §1 горизонт: подкормка → укоренение → офлайн-пространство (этап 03). DS архитектура должна **позволять расширение линейки** без полной переделки.

**Что не берём:**
- Прямая продажа растений (мы — bundle грунта)

#### 2.1.5 Roots — plant-care content как SEO-актив

**Откуда:** [roots-store.ru](https://roots-store.ru/).

**Что берём:**
- **Каталог по нуждам растения** (large / palms / hanging / sets) → у нас будущая фильтрация `/collection` в v2 по «для тёмной комнаты», «для офиса», «для начинающего».
- **Plant care info + Q&A как hub** → у нас `/lab` + `/guide`. Roots делает это как «таксономия видов» (30+ растений) — мы делаем уникально через **научное обоснование с источниками**.

**Что не берём:**
- Корзина и онлайн-checkout
- Loyalty / баллы (наш аналог — Caveat-приписки)

### 2.2 Архитектурные референсы (не эстетические)

| Бренд | Что берём | Где живёт у нас |
|---|---|---|
| **Bloomscape** | Repotting Toolkit как отдельная страница; care guides по растениям | Шаблон `/collection/[plant]` |

### 2.3 3 главных синтез-insight'а

1. **Aesop + Diptyque + MMOMA сходятся в одной точке:** «типографика как brand asset, без украшательств, минимальные акценты, материал/heritage как фон, спокойная навигация». Это прямо тот язык, которым говорят §01 ДНК и §03.5 Apothecary в нашем брендбуке. **Подтверждение направления.**

2. **Krapiva показывает:** фактуры материала работают на бренд сильнее, чем плоские пастельные фоны. Земляная палитра §04 (`soil`, `ceramsite`, `pumice`, `sand`, `gravel`, `chalk`, `graphite`) — **не задействована в вебе**. DS поднимает её как фоновую подложку.

3. **Roots показывает:** plant-care content (`/lab` + `/guide`) — главный SEO-актив plant-бренда. Наш язык — **научное обоснование с 80+ источниками**, не «лайфхаки».

## 3. Этап 3 · Ресёч теории (выжимка)

### 3.1 Spacing — 8pt grid

База 8 px. Шкала: **4 · 8 · 12 · 16 · 24 · 32 · 48 · 64 · 96 · 128**. Соответствует Tailwind `space-1 ... space-32`.

**Правило для DS:** запрещены произвольные значения 36 / 56 / 72 (которые встречаются в §13 текущих веб-мокапах). Только из шкалы.

### 3.2 Типографика — Modular Scale ×1.25 (Major Third)

База `--text-base` = 16px. Шкала Major Third (×1.25) — **самая editorial из распространённых**, соответствует музейной/апотекарной тишине:

```
12  ← caption (xs)
14  ← small (sm)
16  ← body (base) · 1rem
20  ← lead (lg)
24  ← H4 (xl)
30  ← H3 (2xl)
38  ← H2 (3xl)
48  ← H1 (4xl)
60  ← compressed-hero (5xl) · ритуал 00/01/02/03
76  ← display (6xl) · wordmark, hero растения
```

Hero-заголовки на главной — `clamp(2.5rem, 5vw, 4.75rem)` для плавного скейла mobile→desktop.

### 3.3 Токены — Semantic + Primitive

Двухуровневая система:

**Primitive** (из §04 палитры): `--bone`, `--charcoal`, `--moss`, `--soil`, `--ceramsite`, `--pumice`, ...

**Semantic** (12–15 ролей, поверх primitive):

```
--surface-paper        → bone           · основной фон страниц
--surface-deep         → charcoal       · dark-секции
--surface-warm         → chalk          · альтернативный тёплый фон
--surface-soil         → soil/8% opacity· фоновая подложка с фактурой
--text-display         → charcoal       · H1-H2
--text-narrative       → charcoal/0.7   · body
--text-secondary       → charcoal/0.5   · подписи, captions
--text-caption         → charcoal/0.4   · мета, hints
--text-latin           → moss           · латынь (Spectral italic)
--accent-fleuron       → moss           · ❦, акценты
--accent-active        → moss           · hover, focus states
--accent-sku           → CSS var()      · только на странице конкретного SKU
--border-divider       → charcoal/0.3   · линии-разделители
--border-frame         → charcoal/0.5   · apothecary рамки
```

Это даёт: брендбук правит primitive, всё остальное автоматически правится.

### 3.4 Atomic Design (Brad Frost)

```
ATOMS         Button · Input · Label · Badge · Divider · Fleuron · KickerHeader · SerialNumber
              · Doodle · CaveatNote · Frame2x · MaterialDot

MOLECULES     FormField · ProductCardMini · BadgeGroup · SectionHeader · RitualStep
              · WhisperPhrase · ApothecaryBar · Hero · CollectionMini

ORGANISMS     SiteHeader · SiteFooter · ProductCardFull · CollectionGrid · WhisperHero
              · MaterialsGrid · RitualSequence · EmailSubscribeForm

TEMPLATES     /  · /collection · /collection/[plant] · /lab · /guide · /about · /dnevnik

PAGES         (инстансы templates с реальным контентом)
```

### 3.5 Motion — Aesop «elegant motion» as principle

- **Easing**: `cubic-bezier(0.4, 0, 0.2, 1)` (Material standard)
- **Duration**: 200–300ms (UI), 400–600ms (page transitions)
- **Что анимируется**: opacity, translateY (5–10px), border-color, scale (max 1.02)
- **Что НЕ анимируется**: font-size, color (резкие смены), layout properties

Motion усиливает музейность, а не «оживляет» страницу как маркетплейс.

### 3.6 Радиусы / тени / границы

- **Radius**: **острые углы 0–2px max** — критический отказ от mainstream-веба. Острые = музей/heritage; скруглённые = дружелюбный SaaS.
- **Shadows**: **запрещены**. Вместо drop-shadow — тонкие линии (§18 opacity 0.3 для dividers, 0.5 для frames).
- **Borders**: одинарная hairline 0.5–1px, opacity 0.3–0.5. Двойные — только в apothecary frame.

### 3.7 Breakpoints

Tailwind дефолтные: `sm 640 / md 768 / lg 1024 / xl 1280 / 2xl 1536`. Mobile-first.

### 3.8 Доступность

- charcoal/bone = 15.51 (AAA) — везде ОК
- moss/bone = хорошо — текст + accent OK
- SKU-цвета — **только на large text 18pt+** или для декоративных не-текстовых элементов (дудлы, разделители)

### 4.3 Что DS закроет в текущих веб-мокапах §11/§13

| Ошибка в §11/§13 | Решение в DS |
|---|---|
| «В корзину», «корзина · 0» | Кнопки `Button.primary` = «Купить на Ozon» / «Подробнее» / «Подписаться» |
| Бейджи «хит / new / last chance» | `Badge` атом — только функциональные: «N° 001», «1.2 л», «monstera» |
| Hero чёрный + 6-цветная SKU-стрипа | `Hero.default` = bone-фон + Lvl 3 слоган + Caveat-стрелка. Альтернатива — `Hero.Whisper` (Aesop-style) |
| Карточки с буквами M/A/A/С | `ProductCard` = cosmos / iris / SKU-дудл (готовые SVG из `doodles/colored/`) — крупно, на bone-фоне |
| KPI-блок «11 / 1 / 15 / ∞ кайф» разноцветно | Удаляем. Вместо — текст «7 растений · 16 рецептур · 80+ источников» с минимальной типографикой |
| Сердечко ♡ в Caveat | Удаляем (TOV §1.3 — эмодзи никогда). Caveat-приписка без эмодзи |
| Цветной баннер «СПРАВОЧНИК ГРУНТОВ» moss | На bone-фоне + KickerHeader «COLLECTIO · ЛАБОРАТОРИЯ» + H1 «11 компонентов» |
| Произвольные размеры | Modular scale Major Third — все размеры из шкалы |
| Скруглённые углы | Все компоненты `rounded-none` (или max `rounded-sm` для inputs) |
| Drop-shadows | Удаляем все. `Divider` + `Frame2x` вместо |

### 4.4 Что DS добавит сверх брендбука

| Слой | Что добавляем |
|---|---|
| **Semantic tokens** | 12–15 семантических ролей поверх primitive (отсутствуют в брендбуке) |
| **Atomic design структура** | atoms / molecules / organisms / templates (брендбук не структурирован так) |
| **Modular type scale** | Major Third ×1.25 (брендбук даёт веса, но не шкалу размеров) |
| **8pt spacing grid** | Шкала 4...128 (брендбук даёт принципы, но не явную шкалу spacing) |
| **Motion-правила** | Easing, duration, что анимируется (брендбук молчит про motion) |
| **Состояния компонентов** | Default · Hover · Focus · Active · Disabled · Loading · Error (брендбук показывает только дефолтные) |
| **Тёмная подложка для секций** | Использование земляной палитры (`soil`, `ceramsite`, `chalk`, `graphite`) как фактурного фона — Krapiva insight |

### 4.5 Метод проверки на этапе 12 (консистентность)
