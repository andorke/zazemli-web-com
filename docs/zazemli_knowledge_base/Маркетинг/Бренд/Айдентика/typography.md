## Роль в ДНК-четвёрке

Три шрифтовых семейства разнесены по источникам ДНК-четвёрки (см. `dna.md` §1):

| Семейство | Источник ДНК | Что несёт |
|---|---|---|
| **Unbounded** | 02 «Фармацевт» | Аптекарская типографика как главный brand asset. Крупные буквы, уверенный ритм, шрифт ведёт композицию. |
| **Spectral italic** | 02 «Фармацевт» | Латынь рода (`monstera`, `spathiphyllum`) — лабораторный протокол описания. |
| **Caveat** | 04 «Друг» (форма) | Рукописная наивная нота, типографическая сторона источника 04. Парная вербальная сторона (голос Насти) живёт в `voice-principles.md`. |

Aesop / Diptyque — живой синтез источников 01+02 (галерея + аптека) — это типографические референсы для Unbounded и Spectral.

Caveat-приписки физически принадлежат **мосту 02↔04** (см. `dna.md` §4): рукописная форма (04) играет роль апотекарных виньеток на медальонах, листовках, упаковке (02) — делает аптеку личной.

## Три семейства

| Семейство | Назначение | Где живёт |
|---|---|---|
| Unbounded | Системная типографика бренда | Wordmark, hero, заголовки, body, UI, кнопки, навигация |
| Spectral | Нарративный голос | Латынь (italic), narrative-параграфы в `/about` и `/dnevnik`, fleuron, цитаты |
| Caveat | Рукописная наивная нота | Caveat-приписки, стрелочки, маргиналии «от руки» |

Принцип: один шрифт никогда не делает работу другого. Unbounded не имитирует Spectral italic. Caveat не используется как заголовок. Spectral не идёт в навигацию.

## Type-scale — Major Third ×1.25

База `--text-base = 16px`. Шкала ×1.25 — «editorial»: даёт большие шаги между уровнями и хорошо читается на холодных фонах bone.

| Токен | px | rem | Где применяется |
|---|---|---|---|
| `xs` | 12 | 0.75 | Captions, метки, kicker |
| `sm` | 14 | 0.875 | Small UI, footnotes |
| `base` | 16 | 1.0 | Body, UI-кнопки, навигация |
| `lg` | 20 | 1.25 | Lead-параграф |
| `xl` | 24 | 1.5 | H4 / H2 страниц |
| `2xl` | 30 | 1.875 | H3 |
| `3xl` | 38 | 2.375 | H2 / нижний край body-hero |
| `4xl` | 48 | 3.0 | H1 / верхний край body-hero |
| `5xl` | 60 | 3.75 | Compressed-hero — цифры ритуала 00/01/02/03 |
| `6xl` | 76 | 4.75 | Display — wordmark в hero, имя растения |

Hero главной страницы использует fluid clamp: `clamp(2.5rem, 5vw, 4.75rem)` — для плавного скейла mobile → desktop.

## Letter-spacing — шкала §18

| Контекст | Значение | Применение |
|---|---|---|
| Kicker | 0.18em | COLLECTIO-шапка над H1 |
| Caption-uppercase | 0.25em | Метки в КАПС маленьким кеглем |
| Body | 0 | Дефолт |
| HeadingTight | −0.025em | H1–H2 — сжимаем чуть-чуть для плотности |
| DisplayTight | −0.05em | Display 76 — wordmark, имя растения |
| DisplayExtraTight | −0.075em | Только сверх-крупный display ≥100px (печать, крупные промо) |

В digital применяем все шесть значений по контексту. Промежуточные значения нельзя выдумывать — только из шкалы.

## Правила употребления

1. **Caveat ≤2 на коллекцию.** На одной странице — максимум 2 рукописные приписки. Caveat не используется в заголовках, body, навигации. Это правило защищает источник 04 от размывания — см. `dna.md` §8 Конфликт 2.
2. **Латынь — только род растения.** Botanical name в Spectral italic, цвет moss. Пример: «Monstera deliciosa». Spectral italic не применяется к описаниям, отзывам, любому body-тексту.
3. **КАПС только в логотипе и hero.** Wordmark ЗАЗЕМЛИ — uppercase by design. Hero может использовать КАПС в одной фразе. Заголовки, body, UI — не КАПС. Исключение: `caption-uppercase` для маленьких меток с letter-spacing 0.25em.
4. **Body — Unbounded 400 / 16 / line-height 1.5 / letter-spacing 0.** Никаких альтернатив для UI-копи.
5. **Italic = только Spectral.** Unbounded italic не используется.
