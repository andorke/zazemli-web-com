# Иконография · Версия: 1.0.1 · Дата: 2026-05-14 · Источник: brand-v3 §03.5 + §08 + §18 + DS-research §1.2.3 + §1.2.8 + dna.md §1, §4

В бренде нет иконок в привычном UI-смысле. Их место занимают четыре apothecary-якоря, дудлы, handscript-фразы и fleuron. Это не декор, а несущая графика.

## Роль в ДНК-четвёрке

Иконография — самый «многослойный» foundation: её элементы физически принадлежат **сразу нескольким источникам ДНК** и реализуют большинство мостов между ними (см. `dna.md` §1, §4):

| Элемент | Роль в ДНК |
|---|---|
| Apothecary-якоря (COLLECTIO kicker, N°, Frame2x, ApothecaryBar) | Источник 02 «Фармацевт» — внутренний апотекарный сет, типографический протокол описания объекта. |
| Fleuron ❦ (Spectral, moss) | Синтез 01+02 — галерейная тишина (01) + аптекарский знак (02). |
| Дудлы neutral (charcoal) + handscript SVG | **Мост 02↔04** — рукописная форма как апотекарная виньетка на медальонах, листовках, упаковке. Делает аптеку личной. |
| Дудлы colored (SKU-цвет) | Источник 02 (формат апотекарной маркировки) с SKU-идентификацией на `/collectio/[plant]`. |
| MaterialDot | **Мост 02↔03** — земляная точка (03) внутри апотекарного протокола описания (02): лаб-страница, состав в дневнике, `/collectio/[plant]`. |
| Caveat-приписка в пустом поле | **Мост 01↔04** — тёплое слово (04) в галерейной пустоте (01); приглашение, разрушающее холод музея. |

См. `dna.md` §4 «Мосты между источниками» — иконография — главный инструмент дозировки этих мостов.

## Четыре apothecary-якоря

| Якорь | Что это | Где применяется |
|---|---|---|
| **COLLECTIO kicker** | Шапка-метка над H1 в Unbounded 500 / 11–12pt / letter-spacing 0.18em + Spectral italic с N° и партией | Над каждым H1 страницы. Пример: «COLLECTIO ZAZEMLI · N° 001 · monstera» над H1 `/collectio/monstera`. Атом `KickerHeader`. |
| **N°[партия][SKU]** | Серийный номер в Spectral italic, размер до 36pt, цвет SKU | Декоративный маркер на `/collectio/[plant]`. Атом `SerialNumber`. |
| **2-frame composition** | Двойная рамка: внешняя 1px op 0.5 + внутренняя 0.5px op 0.3, gap 6px, цвет charcoal | Опциональный wrapper карточек и секций. Атом `Frame2x`. |
| **Apothecary-bar** | Разделитель: `hairline op 0.3 · dot · ❦ moss · dot · hairline op 0.3` | Между секциями страниц. Атом `Divider` / `ApothecaryBar`. |

Якоря — мост между печатной партией 0 (медальоны, листовки, дневник) и вебом. Они должны быть видны минимум в четырёх местах любого шаблона.

## Fleuron ❦

| Свойство | Значение |
|---|---|
| Глиф | ❦ (Unicode U+2766) |
| Шрифт | Spectral |
| Цвет | moss `#4A7C59` |
| Opacity | 0.8 |

Применяется в центре apothecary-bar, в конце манифеста на `/about`, как акцент в разворотах `/dnevnik`. Не используется как маркер списка, не дублируется в одной фразе, не появляется в иных цветах.

## Opacity-шкала §18

Десять уровней. Шесть основных — для текста и линий, четыре специальных — для нестандартных контекстов.

| Уровень | Где применяется | Часть шкалы |
|---|---|---|
| 0.8 | Heading display (H1, H2) | Основная |
| 0.75 | Латынь (Spectral italic, moss) | Основная |
| 0.7 | Narrative body (Spectral в `/about`) | Основная |
| 0.5 | Secondary text, captions крупно | Основная |
| 0.4 | Captions, hints, footnote | Основная |
| 0.3 | Dividers, hairlines, inner frame | Основная |
| 0.85 | Caveat-приписки на bone | Специальная |
| 0.55 | Day-labels в `/dnevnik` (12 days) | Специальная |
| 0.06 | Watermark min — фоновые знаки | Специальная |
| 0.2 | Watermark max — фоновые знаки | Специальная |

От чего к чему: 0.8 — самый плотный читаемый текст; 0.06 — почти невидимый знак на bone. Промежуточные значения нельзя выдумывать — только из шкалы.

## Дудлы

В `assets/doodles/` живут два набора:

| Набор | Кол-во | Цвет | Где применяются |
|---|---|---|---|
| `doodles/neutral/` | 14 | charcoal | Hero, фоны секций на любых страницах. Универсальный декор. |
| `doodles/colored/` | 7 | SKU-цвет растения | Только на `/collectio/[plant]` — крупно на bone-фоне, как «визитка» растения. |

Правила:
1. Не больше 4 нейтральных дудлов на странице.
2. Цветной дудл — ровно один на странице (тот, что совпадает с SKU).
3. Дудлы не центрируются строго — они смещены, как маргиналии.
4. Дудлы не используются как иконки в UI (кнопки, инпуты).

## Handscript SVG

9 каллиграфических фраз, выгравированных как SVG (не текст). Применяются как акцент в hero `/about`, в разделах ритуала, в `/dnevnik`. Каждая фраза — самостоятельный визуал, не редактируется текстом в коде. Хранятся в `assets/handscript/`.

Handscript физически принадлежит мосту 02↔04 (см. `dna.md` §4): рукописная форма (04) играет роль апотекарных виньеток на медальонах, листовках, упаковке (02).
