# Voice Principles · Версия: 1.0.1 · Дата: 2026-05-14 · Источник: TOV v1.2 §1, §6, §7 + dna.md §1 (источник 04, вербальная сторона)

## Формула голоса Насти

> Деловая структура — несущая конструкция. Забота — материал, из которого она построена.

Голос бренда «Заземли» = личный авторский голос основательницы Насти. Любой регистр (Наставник / Подруга / Редактор) пропускается через этот фильтр.

## Роль в ДНК-четвёрке

Голос Насти — **вербальная сторона источника 04 «Друг»** из `dna.md` §1. Источник 04 — двусторонний:

| Сторона | Что это | Где живёт |
|---|---|---|
| **Форма** — рукописная наивность | Caveat-шрифт, handscript-фразы, дудлы, ручные стрелки | `01_foundations/typography.md`, `01_foundations/iconography.md` |
| **Смысл** — голос Насти | Шутка-шепот, ирония, подмигивание, личная приписка | Этот документ + `founder-quotes.md` + `word-list.md` |

Без формы голос растворится в строгости источника 02 «Фармацевт». Без смысла рукописное превратится в декор без авторства. Только вместе — Caveat-приписка как «след автора».

Голос Насти **пронизывает все три регистра** (Наставник / Подруга / Редактор), не только Caveat-приписки. В ДНК он живёт именно как роль «друга», тёплое микро-касание.

## 6+1 атрибутов голоса

1. **Современный** — никакого апотекарства XIX века в речи. Апотекарство живёт в визуале (рамки, fleuron, латинские роды). Текст — язык сегодняшнего дня.
2. **Деловой** — суть → факт → действие. Без подводок «знаешь, я тут подумала». Если слово можно убрать — убираем.
3. **Цифровой / айтишный** — естественные метафоры из IT (zoom, дашборды, продукты, деплой, спринт). Не нарочито — родной язык.
4. **Эстетичный** — за каждой буквой стоит вкус. Фраза «работает, но некрасиво» переписывается. Эстетика на уровне ритма, не только смысла.
5. **С иронией** — лёгкой, на полшага. Не сарказм, не самоуничижение. Ирония умного человека, который знает себе цену.
6. **С женственной теплотой через заботу** — не «как я тебя понимаю», а тепло через действие: «подготовили для тебя», «в коробке всё по шагам», «положим открытку».
7. **Всё структурирует** — списки, нумерация, таблицы, чёткие заголовки. Это не сухость — это уважение к времени читателя.

Бенчмарки баланса: Aesop (структура + материальность), Diptyque (минимализм + чувственность), Erbology (точность + забота о теле).

## 4 правила брендбука §06

| Правило | Что значит |
|---|---|
| **Строгость + подмигивание** | Деловая структура с лёгкой иронией. Никогда — только одно из двух. |
| **Наивность ≠ инфантильность** | Простой язык, но не сюсюкающий. «Заземли монстеру» — да; «растеньице» — нет. |
| **Ритуал ≠ рутина** | Пересадка — это ритуал. Не «уход за цветочками», не «домашние хлопоты». |
| **Друг ≠ продавец** | Никаких «Купи прямо сейчас!», «Спасибо за выбор!». Тон — знакомой из IT, не менеджера. |

## 3 регистра кратко

| Регистр | Суть | Пример |
|---|---|---|
| **Наставник** | Деловой айтишник + тёплая внимательность. Тех-док, написанный человеком, который хочет, чтобы у тебя получилось. | «Под антуриум — отдельная смесь. Кора, кокос, мох.» |
| **Подруга** | Деловая знакомая из IT со вкусом к заботе. Иронизирует на летучке, но коробку упаковала вручную. | «Бокс заземлён. Открой, разложи, посади.» |
| **Редактор** | Эстетичный минималист с тихой женственностью. Пауза, ритм, вкус. | «Заземли свою монстеру.» |

Подробности и матрица применения — см. `register-matrix.md`.

## Технические правила

| Правило | Значение |
|---|---|
| Имя продукта | «бокс» (не коробка, не набор) |
| Обращение | на «ты», без «Вы», «уважаемый клиент» |
| Эмодзи | никогда, нигде |
| Длина | сократи на 30% после первого черновика |
| Латынь | только род растения (monstera, spathiphyllum). Декоративная — запрещена |
| Caveat-приписки | максимум 2 на всю коллекцию |
| Типографика | Ё всегда Ё. Тире длинное «—». Кавычки «ёлочки». |
| Восклицательный знак | минимум, лучше точка |
| КАПС | только в логотипе и Hero-заголовках |
