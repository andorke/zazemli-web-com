# Tone Matrix · Версия: 1.0.0 · Дата: 2026-05-13 · Источник: TOV v1.2 §9, §10

Единая навигационная матрица: какой регистр, какая визуальная плотность и какой тон обучения применяются в каждом канале и касании. Регистры подробно — см. `register-matrix.md`. Жанры — `genre-templates.md`.

## Принцип трёх каналов

| Канал | Время на «обучение» | Тон обучения |
|---|---|---|
| **Ozon карточка** | 3–5 секунд на первый экран | НЕ обучаем — переводим. Только понятная польза и формат. Никакого манифеста и ритуала. |
| **Instagram** | 15–60 секунд на Reels, длинно в постах | ОБУЧАЕМ. ASMR-замешивание, до/после, разбор «почему эпипремнуму нужен другой грунт», манифест-посты раз в месяц. Главный канал обучения. |
| **Сайт zazemli.com** | бесконечность | ГЛУБОКО ОБУЧАЕМ. Лаб-страница про 11 компонентов, гайд по пересадке, философия. Сюда падают QR-сканы — здесь Настя достраивает картину уже купившему. |

## Объединённая матрица: касание × регистр × плотность × тон обучения

| Касание | Канал | Регистр | Визуальная плотность | Тон обучения | Пример |
|---|---|---|---|---|---|
| Карточка товара | Ozon | Наставник | плотная: буллеты, цифры, SKU | НЕ обучаем, переводим | «Бокс для пересадки антуриума. Под горшок 1.2 л.» |
| Инструкция в коробке | внутри бокса | Наставник + Подруга (◐) | средняя: нумерованные шаги | пошагово обучаем процессу | «Что подготовить заранее: 1. Пустой горшок» |
| IG-пост / Reels | Instagram | Подруга | разреженная: 4-6 строк, воздух | обучаем через личное проживание | «сегодня заземляли антуриум» |
| Push после доставки | мобайл | Подруга | минимум: ≤30 + ≤80 знаков | не обучаем, фиксируем момент | «Бокс заземлён. Открой, разложи, посади.» |
| Welcome email | email | Подруга → Наставник | средняя: список + подпись | подготовка к ритуалу | «Бокс едет. 3 минуты подготовки — пока ждёшь.» |
| Манифест | сайт, упаковка | Редактор | воздушная: списки исключены | не обучаем, задаём принцип | «Заземли свою монстеру.» |
| Страница «О бренде» | сайт | Редактор | воздушная + цитаты основателя | глубоко обучаем философии | «руки в земле — голова свободна» |
| Лаб-страница | сайт | Наставник | плотная: компоненты, проценты, роль | глубоко обучаем рецептуре | «11 компонентов · кора, кокос, мох» |
| Упаковка / коробка | физика | Редактор + Подруга (◐) | минимум: 1-2 фразы на поверхности | задаём ритуал | «Заземлись · пересади · выдохни» |
| Ответ в Direct | IG | Подруга | разговорная: 1-3 строки | отвечаем по ситуации | «Да, поняла. Под спатифиллум подойдёт M.» |
| Ответ на негативный отзыв | Ozon, IG | Подруга | средняя: факт + действие | кризисный сценарий, см. crisis-playbook | «Промах — давай разберём.» |
| Ответ на позитивный отзыв | Ozon, IG | Подруга | минимум: благодарность без сюсюкания | фиксируем диалог | «Записали тебя в коллекцию апреля.» |
| Открытка от руки | физика | Подруга | минимум: одна фраза | жест заботы | «дождётся из отпуска» |
| Пресс-релиз | внешний | Редактор + Наставник (◐) | средняя: цитата + факты | обучаем через цитаты основателя | цитата 1, цитата 2 из `founder-quotes.md` |

## Условные обозначения

- основной регистр = выделен жирным или указан первым
- допустимый регистр = пометка ◐
- визуальная плотность: плотная (буллеты, таблицы) / средняя (списки + абзацы) / разреженная (короткие абзацы) / воздушная (без списков, ритм)

## Практические следствия

- Тексты для Ozon — Наставник, плотно, без поэтики и манифеста.
- Тексты для IG — Подруга, обучающие через личное проживание ритуала.
- Тексты для сайта — Редактор + Наставник в зависимости от страницы (Манифест = Редактор, Лаб-страница = Наставник).
- На упаковке встречаются Редактор и Подруга — короткие фразы, не абзацы.

## Где это работает

- `register-matrix.md` — описание каждого регистра отдельно
- `genre-templates.md` — шаблоны жанров под эту матрицу
- `voice-principles.md` — формула голоса под все клетки матрицы
- `crisis-playbook.md` — детализация для кризисных касаний
- `founder-quotes.md` — что вставлять в касания с Редактором
