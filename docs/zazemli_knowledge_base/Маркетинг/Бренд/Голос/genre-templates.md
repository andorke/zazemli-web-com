# Genre Templates · Версия: 1.0.0 · Дата: 2026-05-13 · Источник: TOV v1.2 §4

Шесть рабочих шаблонов под основные жанры копирайта «Заземли». Каждый шаблон: формат + регистр + длина + примеры. Лексика — см. `word-list.md`. Регистры — см. `register-matrix.md`.

## 1. Карточка Ozon

- **Формат:** заголовок + подзаголовок + буллеты + финал
- **Регистр:** Наставник
- **Длина:** заголовок ≤60 знаков, описание плотное, без поэтики

**Структура:**

```
Заголовок (≤60): «Бокс для пересадки [растения]. Под горшок [N] л.»

Подзаголовок: «Внутри — [рецептура под род] + дренаж + корневин + перчатки + дневник на год.»

Буллеты:
— [N] компонентов в смеси · [процент] [главный компонент]
— под объём горшка [N] л — точно столько, сколько нужно
— для [растения] — а не «для всего сразу»
— дневник, чтобы не забыть, когда полить

Финал: «Заземли [растение]. Дальше — наблюдай.»
```

**Пример заполненный:**

> Бокс для пересадки антуриума. Под горшок 1.2 л.
> Внутри — рецептура из 11 компонентов под ароидные + дренаж + корневин + перчатки + дневник на год.
> Заземли антуриум. Дальше — наблюдай.

## 2. IG-подпись к Reels

- **Формат:** 4 строки, без эмодзи, без хэштегов в тексте
- **Регистр:** Подруга
- **Длина:** 4–6 строк, не больше

**Структура:**

```
1 строка — крючок («сегодня заземляли антуриум»)
1 строка — что показала
1 строка — деталь / шутка про себя
1 строка — call к диалогу или к Ozon
```

**Пример:**

> сегодня заземляли антуриум — он давно просил
> сняла, как замешиваю смесь: кора, кокос, мох
> руки в земле час, голова — впервые за неделю — пустая
> бокс на Ozon, ссылка в шапке

## 3. Push после доставки

- **Формат:** заголовок ≤30, тело ≤80
- **Регистр:** Подруга
- **Длина:** одна строка заголовка + одна строка тела

**Структура:**

```
Заголовок (≤30): «Бокс заземлён.»
Тело (≤80): «Открой, разложи, посади. Если будет сторис — отметь @zazemli.»
```

## 4. Welcome email

- **Формат:** subject + приветствие + нумерованный список + подпись
- **Регистр:** Подруга → Наставник (внутри одного письма)
- **Длина:** короткое, без воды

**Структура:**

```
Subject: «Бокс едет. 3 минуты подготовки — пока ждёшь.»

Привет.

Что подготовить заранее:

1. Пустой горшок [N] л
2. Старая газета или клеёнка (земля будет везде)
3. Стакан воды (полить после)

Растение готовить не нужно — бокс приедет, в дневнике всё по шагам.

— Заземли
```

## 5. Манифест / «О бренде»

- **Формат:** открывающее предложение + 2-3 коротких абзаца + закрытие
- **Регистр:** Редактор
- **Длина:** короче, чем кажется нужным. Воздух важнее текста

**Структура:**

```
Открывающее предложение (4–8 слов): «Заземли свою монстеру.»
Развитие (2–3 коротких абзаца): про ритуал, про паузу, про себя
Закрытие (1 строка): возврат к глаголу-имени
```

**Особенность:** в Манифесте можно отказаться от списков и таблиц ради воздуха. Это исключение из правила структурности.

## 6. Ответ в Direct

- **Формат:** подтверждение + ответ + приглашение
- **Регистр:** Подруга
- **Длина:** ровно столько, сколько нужно для ответа

**Структура:**

```
1. Подтверждение, что услышала: «Да, поняла.»
2. Конкретный ответ.
3. Если уместно — приглашение продолжить: «Если ещё что — пиши.»
```

**Пример:**

> Да, поняла. Под спатифиллум подойдёт наш бокс M — там смесь рыхлее, ароидным так лучше. Если ещё что — пиши.

- `tone-matrix.md` — какой шаблон в каком канале
- `crisis-playbook.md` — отдельные шаблоны для кризисных ответов
