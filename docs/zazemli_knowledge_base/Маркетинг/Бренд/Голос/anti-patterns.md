# Anti-patterns · Что НЕ применять

Версия: 1.1.0 · Дата: 2026-05-14 · Источники: dna.md §1, §2, §8 + brand-v3 §11 + §13 (анти-эталон) + DS_RESEARCH §1.3 + TOV v1.2 §3
Назначение: разблокировать refactor сайта (Иван, СТО). Источник правды для веба — DS v1.0 + tokens.json. Этот документ описывает только то, что НЕ применять.

---

## 0. Главный принцип

Брендбук `brand-v3.html` — двухскоростной документ. Это зафиксировано в DS_RESEARCH §1.1 и в плане архитектуры артефактов v1:

- Печатно-апотекарная часть `brand-v3` уже **декомпозирована** в самостоятельные артефакты:
  - §01 ДНК → `02_voice/dna.md` (источник правды по ДНК-четвёрке)
  - §02 слоган-матрёшка → `02_voice/slogan-matrix.md`
  - §03.5 Apothecary + §08 дудлы → `01_foundations/iconography.md`
  - §04 палитра → `01_foundations/color-system.md`
  - §05 типографика → `01_foundations/typography.md`
  - §18 правила-токены → `01_foundations/tokens.json`
  - §07 Editorial Whisper, §15 ритуал — пока в `brand-v3` (ждут переезда в `03_components/molecules/`)
- Веб-применения `brand-v3` (§11 UI-компоненты, §13 Мокапы страниц) — **анти-эталон**. Эта часть говорит на чужом языке (e-commerce-маркетплейс) и нарушает собственные правила бренда из §04 и TOV v1.2 §1.3.

Правило приоритета (применять всегда, на любой конфликт):

1. DS v1.0 + `tokens.json` побеждают всё остальное.
2. Декомпозированные артефакты (`dna.md`, `slogan-matrix.md`, `color-system.md`, `typography.md`, `iconography.md`) — источник принципов.
3. Печатно-апотекарная часть `brand-v3` (§07, §15, и оставшиеся ещё-не-декомпозированные секции) — переходный источник, до их переезда в новые артефакты.
4. §11 и §13 `brand-v3` — анти-эталон. Использовать только как «как НЕ надо».

Каждая страница сайта подчиняется главному принципу DS (DS_RESEARCH §4.1): «Каждая страница — экспонат в музее. Лишнее не входит. Тишина дороже громкости. Типографика говорит, цвет молчит, кроме momentaries SKU-акцентов на странице конкретного продукта.»

## 1. Анти-паттерны визуала (digital)

10 пунктов ниже — расширенная версия таблицы из DS_RESEARCH §1.3. Каждый пункт: что нарушает в `brand-v3`, какое правило нарушено, куда смотреть в DS v1.0 за решением.

### 1.1 Кнопки и микро-копирайт e-commerce-склада
- Что: §11 UI «карточка товара» — кнопка «в корзину», бейдж «корзина · 0», пиктограмма корзины.
- Нарушение: у бренда нет корзины — продажа через Ozon (TOV §10). Лексика «корзина / оформите заказ» из blacklist TOV §3.
- Вместо: `Button.primary` с тремя CTA — «Купить на Ozon», «Подробнее», «Подписаться» (DS v1.0 §4.1). Шапка без корзины и счётчика.

### 1.2 Бейджи «хит / new / last chance»
- Что: §11 product-card с маркером «хит» / «new» / «last chance».
- Нарушение: маркетплейс-лексика, регистр Редактора (TOV §1.2), нарушение 60/30/10.
- Вместо: атом `Badge` (DS v1.0 §4.5) — только функционально: «N° 001», «1.2 л», «monstera». Без оценочных меток.

### 1.3 Hero чёрный с CAPS-набором и 6-цветная SKU-стрипа
- Что: §13 «Главная» — hero на чёрном фоне, «ЗАЗЕМЛИ» в Unbounded caps 56px, стрипа из 6 SKU-цветов.
- Нарушение: 60/30/10 (см. `color-system.md`). SKU-цвета — 0% по умолчанию, только на `/collectio/[plant]`. КАПС в потоке запрещён (TOV §3 + §6).
- Вместо: `Hero.default` — bone-фон, слоган Lvl 3 «Заземли растение, заземли себя.» (см. `slogan-matrix.md` §2), Caveat-стрелка. Альтернатива — `Hero.Whisper` (DS v1.0 §6.2).

### 1.4 Карточки с буквами-плейсхолдерами и цветной 2px-полосой
- Что: §13 каталог растений плитками «M», «A», «С» по первой букве + цветная полоса 2px снизу.
- Нарушение: эстетика дешёвого каталога, чужая ДНК (см. `dna.md` §1 — конфликт с источником 01 «Галерист» и источником 02 «Фармацевт»). Буква не показывает ни растения, ни рецептуры.
- Вместо: молекула `ProductCardMini` (DS v1.0 §5.1) — крупный SKU-дудл из `doodles/colored/` на bone-фоне (см. `iconography.md`), KickerHeader «N° 001 · monstera» сверху, без цветных полос.

### 1.5 KPI-блок с цветными цифрами
- Что: §13 блок «11 / 1 / 15 / ∞ кайф», каждая цифра в своём SKU-цвете.
- Нарушение: 60/30/10 + антитеза музейности (источник 01 «Галерист»). Музей не делает разноцветный счётчик посещений.
- Вместо: текстовый блок «7 растений · 16 рецептур · 80+ источников» в минимальной типографике (DS v1.0 §6.3 CollectionPreview).

### 1.6 Сердечко и эмодзи в Caveat-приписке
- Что: §13 Caveat-фраза с символом «♡».
- Нарушение: TOV §1.3 + §6 — «Эмодзи — никогда, нигде». Брендбук собственным веб-мокапом нарушает собственное правило. Также — разрушает источник 04 «Друг» в его dna-формулировке (форма + смысл; эмодзи — заменитель смысла без авторства).
- Вместо: `CaveatNote` (DS v1.0 §4.3) без эмодзи и декоративных символов. Максимум 2 приписки на всю коллекцию (TOV §1.3, см. `typography.md` правило 1).

### 1.7 Цветной баннер на moss-фоне
- Что: §13 «Лаборатория грунтов» — широкая полоса moss во весь экран с «СПРАВОЧНИК ГРУНТОВ».
- Нарушение: moss как сплошной фон превращает страницу в эко-маркет. Moss = 10% accent (см. `color-system.md`), не surface.
- Вместо: bone-фон + KickerHeader «COLLECTIO · ЛАБОРАТОРИЯ» + H1 «11 компонентов» + ApothecaryBar (DS v1.0 §4.4 + §6 для `/lab`).

### 1.8 Произвольные размеры шрифта вне шкалы
- Что: §13 использует 56 / 32 / 80 / 22 / 18 / 28px без системы.
- Нарушение: нет modular scale. См. `typography.md` Type-scale: шкала Major Third ×1.25 — 12/14/16/20/24/30/38/48/60/76.
- Вместо: только из шкалы (`tokens.typography.scale`). Hero — `clamp(2.5rem, 5vw, 4.75rem)`.

### 1.9 Скруглённые углы и drop-shadows
- Что: §11 кнопки и карточки с radius 4-8px и drop-shadow.
- Нарушение: DS_RESEARCH §3.6 — острые углы (0-2px), shadows запрещены. Скруглённые читаются как дружелюбный SaaS; нужен музей (источник 01 «Галерист»).
- Вместо: `rounded-none` или `rounded-sm` (только form-inputs). Вместо теней — `Divider`, `ApothecaryBar`, `Frame2x` (см. `iconography.md`).

### 1.10 Земляная палитра не задействована как фактура
- Что: §13 использует только bone как фон. Earth-тона из `color-system.md` (soil/ceramsite/pumice/sand/gravel/graphite) на веб не выведены.
- Нарушение: источник 03 «Натуралист» исчезает из веба (см. `dna.md` §1, §8 Конфликт 1). Krapiva-insight DS_RESEARCH §2.1.4: материал как фактурная подложка работает сильнее, чем плоский фон.
- Вместо: `--surface-soil` (semantic-токен, DS v1.0 §3) — фактурная подложка с opacity 8% для отдельных секций. Особенно `/lab` и блоки «что в боксе» на `/collectio/[plant]`. **Но**: не как сплошной фон секции — только локально (см. `dna.md` §8 разрешение конфликта).

## 2. Анти-паттерны типографики

### 2.1 КАПС в потоке абзаца
- Источник запрета: TOV §3 + §6. КАПС только в логотипе и Hero на Unbounded.
- Что в brand-v3: §13 «СПРАВОЧНИК ГРУНТОВ», «ЛАБОРАТОРИЯ» в декоративных плашках.
- Вместо: KickerHeader (Unbounded 500 · 11pt · letter-spacing 1.6-2 · см. `iconography.md`) + H1 в нижнем регистре Unbounded 700.

### 2.2 Caveat больше двух раз на коллекцию
- Источник запрета: TOV §1.3 — «максимум 2 на всю коллекцию». См. также `dna.md` §8 Конфликт 2 (Caveat как дозируемый мост 02↔04).
- Что в brand-v3: §13 Caveat-стрелки практически на каждой секции и каждой карточке.
- Вместо: `CaveatNote` (DS v1.0 §4.3) максимум дважды за всю воронку посещения.

### 2.3 Произвольные letter-spacing
- Источник запрета: `typography.md` Letter-spacing-шкала — 6 значений (kicker 0.18em / caption-uppercase 0.25em / body 0 / headingTight −0.025em / displayTight −0.05em / displayExtraTight −0.075em).
- Что в brand-v3: §11 в кнопках и тегах применяет 0.5 / 0.8 / 1.2 / 3 — все вне шкалы.
- Вместо: только 6 значений шкалы (`tokens.typography.letterSpacing`).

### 2.4 Декоративная латынь
- Источник запрета: TOV §1.3 + §3 — «латынь только род растения, декоративная запрещена».
- Что в brand-v3: §13 hero вставляет декоративные латинские фразы как украшение.
- Вместо: латынь — только род растения в Spectral italic (`--text-latin`, DS v1.0 §3). Никаких ex/observatio/ritus.

### 2.5 Подмена Unbounded на системный шрифт в body
- Источник запрета: `typography.md` фиксирует Unbounded 400 как body 16px.
- Что в brand-v3: §11 несколько UI-блоков подгружают системный sans в кнопках и инпутах.
- Вместо: Unbounded 400 везде в UI (DS v1.0 §2.2). Резервный stack — `Unbounded, "Helvetica Neue", Arial, sans-serif`.

## 3. Анти-паттерны палитры

### 3.1 Нарушение правила 60/30/10
- Источник правила: `color-system.md`. 60% bone (фон) · 30% charcoal · 10% moss. SKU-цвета 0% по умолчанию.
- Что в brand-v3: §13 главная, `/lab`, `/about`, `/dnevnik` используют 4-6 SKU-цветов одновременно. Это «главное игнорируемое правило» (DS_RESEARCH §1.2.4).
- Вместо: на главной, `/about`, `/lab`, `/guide`, `/dnevnik` — НЕТ SKU-цветов кроме moss. SKU-цвета только на `/collectio/[plant]` (`--accent-sku`, DS v1.0 §3).

### 3.2 Moss как surface
- Источник запрета: `color-system.md` — moss как accent (10%), не surface.
- Что в brand-v3: §13 «Справочник грунтов» — moss во всю ширину как фон секции.
- Вместо: moss — только в `--accent-fleuron`, `--accent-active`, `--text-latin` (DS v1.0 §3). Surface — `--surface-paper` (bone) или `--surface-deep` (charcoal).

### 3.3 Earth-палитра как сплошной цветной фон
- Источник запрета: `color-system.md` + DS_RESEARCH §2.1.4 + `dna.md` §8 Конфликт 1 — earth работает как фактурная подложка opacity 8% или MaterialDot, не сплошные блоки.
- Что в brand-v3: §13 либо игнорирует earth, либо предлагает как однородный заливочный цвет.
- Вместо: `--surface-soil` = soil 8% с текстурой grain (DS v1.0 §3). Сплошной заливочный earth — запрещён.

### 3.4 Цвет как «оживляж»
- Источник запрета: главный принцип DS (DS_RESEARCH §4.1) — «типографика говорит, цвет молчит». Соответствует источнику 02 «Фармацевт» из `dna.md` §1.
- Что в brand-v3: §13 KPI-блок, многоцветная стрипа hero, разноцветные иконки — всё с задачей «оживить».
- Вместо: оживление — через типографику и motion (DS v1.0 §2.7), не через цвет. Цвет = инструмент идентификации SKU.

## 4. Анти-паттерны компоновки и motion

### 4.1 Скруглённые углы
- Источник запрета: DS_RESEARCH §3.6 — острые углы 0-2px max.
- Что в brand-v3: §11 — `border-radius: 4-8px` на всех кнопках и карточках.
- Вместо: `rounded-none` по умолчанию; `rounded-sm` (2px max) только для form-inputs (DS v1.0 §4.2).

### 4.2 Drop-shadows на карточках
- Источник запрета: DS_RESEARCH §3.6 — shadows запрещены.
- Что в brand-v3: §11 продуктовые карточки — shadow lift на hover.
- Вместо: тонкие линии 0.5-1px opacity 0.3-0.5 (см. `iconography.md` Opacity-шкала) + опциональный `Frame2x` (DS v1.0 §4.6).

### 4.3 Иконки в навигации
- Источник запрета: источник 01 «Галерист» (см. `dna.md` §1) + MMOMA-паттерн (DS_RESEARCH §2.1.3) — типографика как первичная навигация.
- Что в brand-v3: §11 шапка с иконками гамбургера, корзины, лупы, аватара.
- Вместо: навигация на Unbounded 400 без иконок, без caps (DS v1.0 §6.1 SiteHeader). Поиск — отдельный сценарий через текстовую ссылку.

### 4.4 Bento-grid и плотная плиточная компоновка
- Источник запрета: источник 01 «Галерист» (см. `dna.md` §1) — «один объект на постаменте» + главный принцип DS.
- Что в brand-v3: §13 пробует bento-вариант для «11 компонентов» и для каталога.
- Вместо: воздух, один объект в крупном кадре. Максимум 3 колонки на desktop, 2 на tablet, 1 на mobile (DS v1.0 §6 CollectionGrid).

### 4.5 «Маркетплейсные» motion-паттерны
- Источник запрета: DS_RESEARCH §3.5 — anim только opacity + translateY (5-10px), 200-300ms, Material easing.
- Что в brand-v3: §13 hover с увеличением шрифта на 4px, цветовой shift на 200ms, «прыгающие» бейджи.
- Вместо: hover = opacity 1→0.8 + translateY −2px + border-color shift. Без scale больше 1.02, без font-size, без layout shifts.

### 4.6 Sticky-баннер «акция / скидка» и счётчики
- Источник запрета: TOV §3 (восклицания в массе, запрет «Скидка 30% только сегодня»), главный принцип DS.
- Что в brand-v3: §13 sticky-полоса «−30% до конца недели» и countdown-таймер.
- Вместо: ни sticky-плашек, ни таймеров. Промо — отдельной мягкой строкой в footer или в Caveat-приписке без восклицания (TOV §8.1: «До конца недели — мини-цена»).

## 5. Анти-паттерны копирайта (из TOV v1.2 §3 + §6)

### 5.1 Запрещённые слова и обороты
- Источник: TOV §3 blacklist.
- Не используем: «приобретите», «оформите заказ», «закажите», «инновационный», «уникальный», «премиальный» (про себя), «уважаемый клиент», «компания», «бренд», «в кратчайшие сроки», «Спасибо за выбор», «эко-», «органик», «натуральный 100%», «земляне».
- Вместо: глаголы whitelist TOV §2.1 — «заземли / заземлись / пересади / выдохни / руки в землю». CTA — «Купить на Ozon», «Подробнее».

### 5.2 Восклицательные знаки и КАПС в абзаце
- Источник: TOV §3 + §6.
- Не применяем: «Купите!!!», «Скидки!!!», КАПС в потоке текста.
- Вместо: точка по умолчанию. КАПС — только в логотипе и Hero.

### 5.3 Эмодзи и декоративные символы Unicode
- Источник: TOV §1.3 + §6 — «Эмодзи — никогда, нигде».
- Не применяем: эмодзи и символы ♡, ★, ✿, ✓ в копирайте, Caveat-приписках, push, IG.
- Вместо: типографические знаки бренда — тире `—`, ёлочки `« »`, fleuron `❦` (только в декоре от DS), точка-разделитель `·`.

### 5.4 Латынь как декорация
- Источник: TOV §1.3 + §3 — «декоративная латынь запрещена».
- Не применяем: «ex cultura», «observatio», «ritus», «manuscriptum».
- Вместо: латынь — только род растения (`monstera`, `spathiphyllum`) в Spectral italic.

### 5.5 Эмпатия-сюсюкание вместо заботы-через-действие
- Источник: TOV §8.2 матрица «забота-действие vs эмпатия-сюсюкание», см. `crisis-playbook.md`.
- Не применяем: «Как я тебя понимаю», «Душой с тобой», «Спасибо, что выбрали нас», «С теплом и любовью, команда».
- Вместо: забота через действие — «Записали тебя в коллекцию апреля», «Дневник внутри — поможет, если что», «— Заземли» вместо «С теплом, команда».

### 5.6 Маркетплейсные глаголы и обороты
- Источник: TOV §3 + §8.1.
- Не применяем: «Оформить заказ», «Добавить в корзину», «Заказ отгружен», «Купите прямо сейчас!», «Партия 0» в публичной речи.
- Вместо: «Купить на Ozon», «Бокс заземлён. Идёт к тебе», «Коллекция апрель 2026».

## 6. Чек-лист «можно ли это применить?»

Перед тем как поставить элемент на сайт/упаковку/IG — все 8 ответов «да». Если хотя бы одно «нет» — не применяй.

1. Размер шрифта из шкалы Major Third (12/14/16/20/24/30/38/48/60/76)? См. `typography.md`.
2. Spacing из шкалы 8pt (4/8/12/16/24/32/48/64/96/128)?
3. Цвет в палитре `color-system.md`, без нарушения 60/30/10? (SKU-цвета только на `/collectio/[plant]`)
4. Углы острые (0-2px max), нет drop-shadow?
5. Текст без слов из blacklist TOV §3, без КАПСа в абзаце, без эмодзи?
6. На странице видно **минимум 3 из 4 источников ДНК** (`dna.md` §1): Галерист 01 + Фармацевт 02 + Натуралист 03 + Друг 04?
7. Носитель **не сидит только на одном полюсе** оси контраста (`dna.md` §2): премиум/наивное, цифровое/материальное, интеллект/ритуал?
8. Есть ссылка на токен DS v1.0 / компонент DS / артефакт foundations, объясняющий применение?

Дополнительно — для конкретного носителя сверить **доли источников** из `dna.md` §7 «Доминанты по типу носителя». Например, hero главной = 50% Галерист + 35% Фармацевт + 5% Натуралист + 10% Друг.
