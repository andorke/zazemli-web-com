## Blacklist

### Запрещённые слова

| Слово / приём | Почему нет |
|---|---|
| «приобретите», «оформите заказ», «закажите» | продавцовское |
| «инновационный», «уникальный», «премиальный» | про себя — нельзя |
| «уважаемый клиент», «уважаемые покупатели» | дистанция |
| «компания», «бренд» (про нас в третьем лице) | дистанция |
| «в кратчайшие сроки», «обращаем ваше внимание» | канцелярит |
| «Спасибо за выбор!» | продавцовское |
| «эко-», «органик», «натуральный 100%» | юридически невалидно |
| «земляне» как обращение к подписчикам | зафиксировано: НЕТ |
| латынь-декорация (ex cultura, observatio, ritus) | только род растения разрешён |

### Запрещённые приёмы

- восклицательные знаки в массе («Купите!!! Скидки!!!»)
- КАПС в потоке текста (Hero-заголовки в Unbounded — да, КАПС в абзаце — нет)
- эмодзи — никогда, нигде
- тире-дефис вместо длинного тире
- английские кавычки " " вместо ёлочек « »
