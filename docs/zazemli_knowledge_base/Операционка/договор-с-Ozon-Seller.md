# Runbook: Договор с Ozon Seller

**Версия:** 1.0.0
**Дата:** 2026-05-16
**Тип:** Операционный runbook
**Раздел БЗ:** Операционка / Площадки / Ozon

---

## 1. Что это

Пошаговая инструкция по заключению договора-оферты с Ozon Seller и активации кабинета продавца. Для проекта ЗАЗЕМЛИ — D2C-бренд боксов для пересадки комнатных растений.

**Цель:** получить активный кабинет seller.ozon.ru, право загружать SKU и принимать заказы.

**Почему критично:** без активированного договора-оферты карточки товаров **нельзя загрузить**, отгрузка невозможна, запуск продаж заблокирован. Это сейчас главный операционный блокер запуска ЗАЗЕМЛИ.

## 3. Состав (предусловия и шаги)

### 3.1. Предусловия

- **ИП зарегистрировано** (для ЗАЗЕМЛИ — выполнено, начало апреля 2026).
- **Расчётный счёт открыт** (для ЗАЗЕМЛИ — Точка, выполнено).
- **ИНН и ОГРНИП на руках**.
- **Выписка ЕГРИП** (не старше 30 дней — скачать из ЛК ФНС).
- **Паспорт ИП** (разворот с фото).
- **Свидетельство о постановке на учёт в ФНС** (ИНН).
- **Реквизиты расчётного счёта** (банк, БИК, к/счёт, р/счёт).
- **E-mail и телефон ИП** для регистрации кабинета.
- **Логотип бренда** (PNG/SVG, формат под требования Ozon — уточнить в кабинете).
- **Подписан НДА с командой** (если есть подрядчики, имеющие доступ).
- **ЭДО подключен** (Контур, СБИС или эквивалент — обязательно для документооборота с Ozon).

### 3.2. Шаги

**Шаг 1. Регистрация на seller.ozon.ru**
1. Зайти на seller.ozon.ru.
2. Нажать «Стать продавцом».
3. Указать ИНН ИП — система автоматически подтянет данные из ЕГРИП.
4. Указать e-mail (рабочий, не личный) и телефон.
5. Подтвердить через SMS / e-mail.

**Шаг 2. Заполнение профиля магазина**
1. Название магазина (для ЗАЗЕМЛИ — «ЗАЗЕМЛИ» или «zazemli», уточнить с маркетингом).
2. Категория товаров (для ЗАЗЕМЛИ — «Дом и сад» → «Растения и уход за ними»).
3. Юридическая форма — ИП.
4. Адрес склада / самовывоза (если FBS) или подключение FBO.
5. Логотип, обложка, описание магазина (черновые, можно обновить позже).

**Шаг 3. Загрузка документов ИП**
1. Свидетельство о регистрации (лист записи ЕГРИП).
2. Свидетельство ИНН.
3. Паспорт (разворот с фото + прописка).
4. Выписка ЕГРИП не старше 30 дней.
5. Реквизиты р/счёта (банковская справка или скан из ЛК банка).

**Шаг 4. Акцепт оферты**
1. Прочитать договор-оферту (хранится в кабинете → «Договоры»).
2. Принять условия — ставится отметка о согласии.
3. **Важно:** сохранить PDF договора в `Юридические документы/02_Договоры_площадки/Ozon/`.

**Шаг 5. Ожидание активации**
- Срок: 1-3 рабочих дня.
- Статус виден в кабинете → «Профиль» → «Статус верификации».
- Если возникнут вопросы — Ozon связывается по e-mail / в чате поддержки.

## 5. Правила (что готовить заранее)

### 5.1. Финансовая модель

Перед стартом учесть в юнит-экономике (диапазоны, уточнить с {роль: финансист} и {роль: категорийный менеджер Ozon}):
- **Комиссия Ozon:** 22-24% для категории «Растения и уход» (актуальный размер смотреть в кабинете → Тарифы).
- **Эквайринг:** 1-1.5%.
- **Логистика FBO/FBS:** зависит от веса, габаритов, дальности. Для бокса с грунтом 1-3 кг — диапазон уточнить через калькулятор Ozon.
- **Возвраты:** заложить 5-15% в зависимости от категории.
- **Хранение FBO:** считается посуточно от объёма.

### 5.2. Требования к карточкам товара

- Главное фото — белый фон, без текста, минимум 700×700px.
- Доп. фото — 6-8 шт, lifestyle + детали.
- Видео — рекомендуется (Reels-формат до 60 сек).
- Описание — структурированное, с буллетами, без воды.
- Атрибуты — заполнять максимально подробно (влияет на показы).
- Для ЗАЗЕМЛИ — отдельно подготовить шаблон карточки (см. контент-план).

### 5.3. Документы на товар

- **Отказное письмо** на грунт — критично, без него карточка может быть скрыта (см. отдельный runbook).
- **Декларация на упаковку** — уточнить, нужна ли для нашего типа.
- **Сертификат на удобрения** (если используем) — уточнить требования.

## 6. Что НЕ применять

- **Не использовать личный e-mail** для регистрации — только рабочий, привязанный к бренду.
- **Не указывать домашний адрес** как склад — лучше отдельное помещение или фулфилмент.
- **Не игнорировать ЭДО** — без него документооборот не работает.
- **Не принимать оферту, не сохранив PDF** — потом сложно найти конкретную версию.
- **Не запускать продажи без отказного письма** — карточка может быть заблокирована, восстановление 1-2 недели.
- **Не использовать чужие фото** — Ozon снимает с продаж за нарушение прав.
