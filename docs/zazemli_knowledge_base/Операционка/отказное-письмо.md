# Runbook: Заказ отказного письма на сертификацию грунта

**Версия:** 1.0.0
**Дата:** 2026-05-16
**Тип:** Операционный runbook
**Раздел БЗ:** Операционка / Сертификация

---

## 1. Что это

Пошаговая инструкция по заказу отказного письма у аккредитованного органа сертификации. Документ подтверждает, что грунт для комнатных растений в боксе ЗАЗЕМЛИ **не подлежит обязательной сертификации или декларированию** по ТР ТС / ТР ЕАЭС.

**Цель:** получить официальный документ-страховку, который можно предъявить:
- Ozon — если запросят при модерации карточки или при риске скрытия;
- Wildberries / Яндекс Маркет — при подключении в будущем;
- Любой проверяющий орган (Роспотребнадзор, ФАС) — при запросе;
- Партнёр, дилер, точка продаж — при заведении товара.

**Почему критично:** без отказного письма карточки на маркетплейсах могут быть скрыты по требованию площадки (особенно при первой модерации). Восстановление с письмом — 1-2 дня, без письма — может затянуться на 1-2 недели.

## 3. Состав (предусловия, исполнители, материалы)

### 3.1. Кто делает (исполнитель)

**Аккредитованный орган сертификации** — частная организация, у которой есть аккредитация Росаккредитации на работу с ТР ТС / ТР ЕАЭС.

Где искать:
- **Реестр Росаккредитации** (fsa.gov.ru) — официальный список аккредитованных органов.
- **Поверенный по ТЗ / патентный поверенный** — может рекомендовать проверенного партнёра (мы уже работаем с одним по ТЗ ЗАЗЕМЛИ).
- **Профильные форумы продавцов Ozon/WB** — отзывы.
- **Прямой поиск:** «заказать отказное письмо грунт для растений» + сравнить 3-5 предложений.

> Уточнить с {роль: юрист / поверенный по ТЗ} рекомендованный орган.

### 3.2. Что нужно для заказа

1. **Реквизиты ИП:** ИНН, ОГРНИП, юр. адрес, ФИО.
2. **Описание продукта:**
   - Название (ЗАЗЕМЛИ — бокс для пересадки комнатных растений).
   - Состав (11 компонентов — см. `Продукт/sku-system.md`).
   - 7 SKU (по типам растений / биотопам).
   - Фасовка (вес одного бокса, объём).
   - Назначение (исключительно для пересадки комнатных растений в домашних условиях, не для сельскохозяйственного производства).
3. **ОКПД2 код** — для грунта для комнатных растений ориентировочно **20.15.80.000** (удобрения и азотные соединения) или **20.15.79.000** — уточнить с органом сертификации, какой корректнее для нашего продукта.
4. **Этикетка / упаковка** (макет в PDF или фото) — орган проверит соответствие маркировке.
5. **Технические условия (ТУ) или ГОСТ**, по которому изготавливается продукт (если нет своих ТУ — указать ГОСТ-аналог; уточнить с {роль: технолог}).

### 3.3. Сроки и стоимость

- **Срок изготовления:** диапазон **1-2 недели** (5-10 рабочих дней). Иногда экспресс — 2-3 дня за доплату.
- **Стоимость:** диапазон **3 000 — 7 000 ₽** за одно письмо. Уточнить с конкретным органом.
- **Срок действия письма:** обычно 1 год, иногда 3 года — зависит от формата. Уточнить при заказе.

## 5. Правила (что и в какой последовательности делать)

### 5.1. Шаги заказа

1. **Найти 3 органа сертификации**, сравнить цены и сроки.
2. **Направить запрос** с описанием продукта (шаблон вопросов — см. ниже §5.2).
3. **Получить КП** (коммерческое предложение), выбрать подрядчика.
4. **Подписать договор / счёт**, оплатить.
5. **Отправить пакет документов** (реквизиты, описание, фото этикетки, ТУ/ГОСТ).
6. **Дождаться готовности** (1-2 недели).
7. **Получить оригинал** (курьером или почтой) + скан в PDF.
8. **Сохранить в Drive** по пути: `Юридические документы/03_Сертификация/otkaznoe-pismo_grunt_2026.pdf`.
9. **Отметить в реестре документов** срок действия и дату следующего продления.

### 5.2. Список вопросов в орган сертификации

При первом запросе спросить:
- Какой формат отказного письма вы предоставляете (форма 1 / форма 2 / письмо в свободной форме)?
- На какие ТР ТС / ТР ЕАЭС письмо распространяется?
- Срок действия письма?
- Стоимость и срок изготовления?
- Какой пакет документов от меня нужен?
- Какой ОКПД2 код вы рекомендуете для грунта для комнатных растений?
- Можете ли вы подготовить декларацию соответствия, если в будущем понадобится?

### 5.3. Хранение

Готовое письмо хранится в трёх местах:
1. **Drive:** `Юридические документы/03_Сертификация/`.
2. **Локально:** копия в защищённой папке на компьютере.
3. **Физически:** оригинал в офисной папке «Юридические документы → Сертификация».

### 5.4. Что делать, если Ozon запросит

1. Получить запрос в кабинете → ответить в течение 24 часов.
2. Прикрепить PDF отказного письма + сопроводительный текст:
   _«Грунт для комнатных растений не подлежит обязательной сертификации согласно ТР ТС [номер]. Прикладываю отказное письмо от [Орган] № [номер], действительно до [дата]»._
3. Сохранить переписку в `Юридические документы/02_Договоры_площадки/Ozon/perepiska/`.

## 6. Что НЕ применять

- **Не заказывать у неаккредитованного органа** — письмо не будет иметь юридической силы.
- **Не использовать чужие отказные письма** (даже от поставщика грунта) — оно выдано на их ИНН и продукт, не на наш.
- **Не игнорировать срок действия** — обновить за месяц до истечения.
- **Не указывать в описании продукта «удобрение»** — это меняет правовой режим и может потребовать обязательной декларации.
- **Не отправлять Ozon без сопроводительного текста** — модератор может не понять контекст.
