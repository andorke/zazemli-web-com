COO-runbook: что подготовить для запуска первой партии на Ozon FBO. Чек-лист документов, сроков, действий — с приоритетами по риску.

## Что это

Операционный документ для COO-роли (Настя в нашем случае). Отвечает на вопрос: **«что у меня должно быть готово, прежде чем я отгружу первый бокс на склад Ozon, и в каком порядке это делать?»**

Это не стратегия (это — `06_strategy/legal-form.md`) и не общая инструкция по Ozon (документация Ozon Seller). Это **наш персональный чек-лист**: что **уже сделано**, что **критично сделать до отгрузки**, что **рекомендуется как страховка**, что **можно отложить**.

⚠️ **Не юр.консультация.** Это операционная сводка на основании общих знаний регуляторики РФ + требований Ozon. Точные ответы — у поверенного по товарному знаку (он же сделает отказное письмо).

## Состав

### 1. Три уровня готовности — по приоритету риска

**🔴 ОБЯЗАТЕЛЬНЫЙ МИНИМУМ** — без этого Ozon просто не примет товар на склад или карточку не активируют. Блокирующие требования системы.

### 2. 🔴 ОБЯЗАТЕЛЬНЫЙ МИНИМУМ

#### 2.1 Договор с Ozon Seller

| Параметр | Значение |
|---|---|
| **Что это** | Договор между ИП и Ozon Marketplace — позволяет торговать на платформе |
| **Где делать** | Личный кабинет Ozon Seller → «Документы» → «Договор оферты» (акцепт оферты) |
| **Что нужно** | Регистрация ИП ✅ + расчётный счёт ✅ + ИНН + ОГРНИП |
| **Срок** | 1-3 рабочих дня (после регистрации в Ozon Seller) |
| **Стоимость** | 0 ₽ |
| **Действия Насти** | (1) Зарегистрировать ЛК на seller.ozon.ru через ИНН; (2) Загрузить документы ИП; (3) Акцептовать оферту; (4) Дождаться активации |
| **Когда делать** | **СЕЙЧАС** — это блокер всего остального, регистрация может занять несколько дней с верификацией |

#### 2.2 Этикетка на каждом боксе с составом

| Параметр | Значение |
|---|---|
| **Что это** | Обязательная маркировка товара по ТР ТС 022/2011 «Пищевая продукция в части её маркировки» (для непищевых — аналогичные требования через ЗоЗПП ст.10) |
| **Что должно быть на этикетке** | • Наименование продукта («Грунт для пересадки [растения], бокс на 1 пересадку») • Состав (компоненты почвосмеси) • Объём / масса нетто • Производитель (ИП [Фамилия], ОГРНИП) • Адрес производства • Дата изготовления • Срок годности (для грунта — обычно 24-36 мес) • Условия хранения (сухое тёмное место) • ОКПД2 код мелким шрифтом • QR на лаб-страницу (опционально, но красиво) |
| **Статус** | ✅ Базовая этикетка готова в `partia-0` PDF, но **проверить наличие ВСЕХ обязательных полей перед печатью** |
| **Срок** | 1-2 дня (только проверка имеющегося макета) |
| **Стоимость** | Уже учтено в смете типографии |
| **Действия Насты** | Открыть `partia-0` PDF медальонов и листовок → сверить с чек-листом полей выше → если чего-то нет, добавить и переотправить в типографию |
| **Когда делать** | **ДО первой партии-печати** (до конца мая 2026) |

#### 2.3 ОКПД2 код в карточке Ozon

| Параметр | Значение |
|---|---|
| **Что это** | Общероссийский классификатор продукции по видам экономической деятельности. Ozon требует при создании каждого SKU |
| **Какой код для нас** | **20.15.80.140** — «Удобрения смешанные минеральные и органические» (если по торфу-основе) или **20.20.19.000** — «Прочие средства защиты растений и регуляторы роста». Точный код — **уточнить у поверенного или в Ozon Seller-категории при регистрации**. Для грунтов с органикой типичный — 20.15.80.140 |
| **Где смотреть** | classifikators.ru → ОКПД2 → грунт / удобрение / субстрат |
| **Срок** | 30 минут на проверку всех вариантов |
| **Стоимость** | 0 ₽ |
| **Действия Насты** | (1) Найти 2-3 подходящих кода ОКПД2; (2) Спросить у поверенного, какой правильный для «бокса пересадки = грунт + допы»; (3) Использовать один код для всех 16 SKU |
| **Когда делать** | **ДО регистрации первого SKU в Ozon** (т.е. одновременно с договором Seller) |

#### 2.4 Декларация о компонентах (информация о составе для покупателя)

| Параметр | Значение |
|---|---|

### 3. 🟡 НАСТОЯТЕЛЬНО РЕКОМЕНДУЕМОЕ (страховка от скрытия карточки)

#### 3.1 Отказное письмо (Letter of Refusal)

| Параметр | Значение |
|---|---|
| **Что это** | Официальный документ от аккредитованного органа сертификации, подтверждающий, что продукт **НЕ подлежит обязательной сертификации** по ТР ТС / ТР ЕАЭС |
| **Зачем нужен** | Ozon может запросить документы по требованию. Без отказного письма карточку **могут скрыть на 5-10 дней** до предоставления. Если запрос пришёл в высокий сезон — это ~20-50 потерянных продаж |
| **Где заказать** | (1) **У поверенного, который ведёт ТЗ** — самый простой путь, у них партнёрский орган; (2) Самостоятельно через любой аккредитованный орган сертификации (Росаккредитация, реестр) |
| **Что нужно для заказа** | (1) Реквизиты ИП; (2) Описание продукта (Заземли = «бокс для пересадки комнатных растений, содержащий грунт-почвосмесь, дренаж, корневин, перчатки, листовку, дневник»); (3) ОКПД2 код; (4) Этикетка-образец |
| **Срок** | 1-2 недели |
| **Стоимость** | **3-7 тыс. ₽** (одно письмо на все 16 SKU, поскольку продукт = «бокс пересадки») |
| **Срок действия** | Бессрочно (пока продукт и регуляторика не изменятся) |
| **Действия Насты** | (1) Позвонить поверенному, спросить «можете заодно сделать отказное письмо на грунт?»; (2) Если да — оплатить + отправить материалы; (3) Получить PDF; (4) Сохранить в Drive `СД/Юридические документы/Отказное_письмо_2026.pdf` |
| **Когда делать** | **СЕЙЧАС параллельно с договором Seller** — занимает 1-2 недели, лучше иметь готовым к первой отгрузке |

#### 3.2 Запас этикеток с QR (на 110% партии)

| Параметр | Значение |
|---|---|
| **Что это** | Запас наклеек на упаковке (медальоны + штрихкоды Ozon + бонд-стикеры) |
| **Зачем нужен** | (1) Брак в типографии = до 5% потерь; (2) Брак при сборке = до 3%; (3) Дозаказы дорогие и долгие |
| **Что должно быть напечатано в запасе** | • Медальоны 80/100/120 мм × 7 SKU — на 110% партии • Welcome-листовка × 16 SKU — на 110% • Бонд-стикеры — на 110% • Штрихкоды Ozon (печатаются на термопринтере по факту) |
| **Стоимость** | +10% к типографской смете = ~2 200 ₽ |
| **Когда делать** | **При заказе типографии** на первую партию |

#### 3.3 Документация на упаковку (если потребует Ozon)

| Параметр | Значение |
|---|---|
| **Что это** | Сертификат / декларация на материалы упаковки (коробка, дой-пак, конверт, сотовая крафт) — обычно делает производитель упаковки |
| **Действия Насты** | При закупке упаковки у Витон / tvoypaket.ru **запросить копии сертификатов на материалы**. Положить в `СД/Юридические документы/Упаковка_сертификаты/`. Если поставщик не даёт — не критично, но иметь полезно. |
| **Стоимость** | 0 ₽ (если поставщик даёт по запросу) |
| **Когда делать** | При закупке упаковки (май 2026) |

## Правила применения

- **Чек-лист идёт по приоритету риска.** Сначала закрываем 🔴 минимум, потом 🟡 страховку, 🟢 откладываем до Спринта 11+.
- **Один звонок поверенному закрывает 4 пункта** (отказное письмо + ОКПД2 + COLLECTIO + комбинированная заявка). Не звонить по одному вопросу — собрать список и звонить один раз.
- **Регистрация в Ozon Seller — параллельно** с подготовкой документов. Это блокер всего дальнейшего, делать первым.
- **Этикетки сверять до печати**, не после. Переделка макета после печати = +5-10 тыс. ₽ дозаказа.
- **Документы хранить в Drive по структуре §7** — не на рабочем столе, не в почте. Это критично при первой проверке.

- ❌ **Не заказывать полную декларацию о соответствии «на всякий случай»** — это 15-30 тыс. ₽ потерянного бюджета. Заказывать только при реальном запросе Ozon или выходе на WB.
- ❌ **Не пытаться обойтись без ОКПД2** — карточку просто не активируют. Это поле обязательное при регистрации SKU.
- ❌ **Не использовать «универсальную» этикетку для 16 SKU** — каждый SKU имеет свой состав почвосмеси (см. `sku-system.md` §5), это должно быть на этикетке.
- ❌ **Не доверять чёткому ответу нейросети по юр.вопросам** — этот документ ориентировочный, точный ответ всегда у поверенного.
- ❌ **Не откладывать звонок поверенному «на потом»** — критический путь к запуску. От первого звонка до отказного письма ~2 недели, до полноценной декларации (если потребуется) ~4 недели.
