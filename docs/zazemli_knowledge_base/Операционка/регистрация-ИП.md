# Runbook: Регистрация ИП через ФНС-онлайн

**Версия:** 1.0.0
**Дата:** 2026-05-16
**Тип:** Операционный runbook
**Раздел БЗ:** Операционка / Юридическая форма

---

## 1. Что это

Пошаговая инструкция по регистрации индивидуального предпринимателя через онлайн-сервис ФНС (nalog.gov.ru), Госуслуги или через банк-партнёр. Для проекта ЗАЗЕМЛИ — D2C-бренд боксов для пересадки комнатных растений.

**Цель:** получить статус ИП с УСН 15% «Доходы минус Расходы», ОГРНИП, выписку ЕГРИП и возможность открыть расчётный счёт.

**Когда нужен:** при запуске нового бизнеса/направления; как референс при разборе аналогичных кейсов; для архива.

**Статус для ЗАЗЕМЛИ:** ИП зарегистрировано в начале апреля 2026, документ хранится для архива и будущих ситуаций.

## 3. Состав (предусловия и шаги)

### 3.1. Предусловия

- **Паспорт РФ** (разворот с фото + прописка).
- **СНИЛС** (нужен для Госуслуг и автоматической регистрации в ПФР/ФСС).
- **ИНН физлица** (если нет — присваивается автоматически при регистрации).
- **ЭЦП** — нужна для подачи через сайт ФНС без визита; не нужна, если подача через банк или с визитом в МФЦ.
- **Подтверждённая учётная запись Госуслуг** (для онлайн-варианта).

### 3.2. Шаги (3 варианта подачи)

**Вариант А — через банк (рекомендуется для скорости):**
1. Выбрать банк-партнёр (Точка, Тинькофф, Сбер, Альфа и т.п.).
2. Заполнить онлайн-анкету на сайте банка.
3. Менеджер банка приезжает или встреча онлайн с верификацией.
4. Банк подаёт документы в ФНС, открывает расчётный счёт.
5. Срок: 3-5 рабочих дней.

**Вариант B — через nalog.gov.ru с ЭЦП:**
1. Зарегистрироваться в личном кабинете ФНС.
2. Заполнить заявление по форме Р21001.
3. Прикрепить заявление на УСН (форма 26.2-1).
4. Подписать ЭЦП и отправить.
5. Госпошлина не взимается при онлайн-подаче.

**Вариант C — через МФЦ или ФНС лично:**
1. Заполнить Р21001 от руки или в программе ФНС.
2. Оплатить госпошлину (диапазон 800 ₽ — уточнить актуальный размер).
3. Принести паспорт + СНИЛС + заявление в МФЦ или ИФНС.
4. Срок: 3 рабочих дня.

### 3.3. ОКВЭДы для ЗАЗЕМЛИ (черновой список — уточнить с бухгалтером)

Основной:
- **47.91** — Торговля розничная по почте или по информационно-коммуникационной сети (D2C/e-commerce).

Дополнительные (рекомендуется):
- **47.76.1** — Торговля розничная цветами и другими растениями, семенами и удобрениями в специализированных магазинах.
- **47.76.2** — Торговля розничная домашними животными и кормами для домашних животных (на всякий случай — если расширение).
- **47.99** — Торговля розничная прочая вне магазинов, палаток, рынков.
- **82.99** — Деятельность по предоставлению прочих вспомогательных услуг для бизнеса.
- **74.20** — Деятельность в области фотографии (для контента).
- **73.11** — Деятельность рекламных агентств (для маркетинговой активности).

> Уточнить финальный список с {роль: бухгалтер}.

### 3.4. УСН 15% «Доходы минус Расходы»

- Подаётся **одновременно** с заявлением на регистрацию ИП (форма 26.2-1) или в течение 30 календарных дней с даты регистрации.
- Для ЗАЗЕМЛИ выбрана именно эта схема (а не 6% Доходы), потому что закупка сырья для боксов — значимая расходная часть.
- Альтернатива (АУСН, патент) — нужно уточнить с {роль: бухгалтер}.

## 4. Варианты применения

- Регистрация нового ИП на основное направление.
- Регистрация ИП для пилота / второго бренда (если будет).
- Передача инструкции коллеге / партнёру.
- Использовать как чек-лист перед общением с банком.

---

## 5. Правила (что делать в первые 7 дней после регистрации)

1. **День 1-2:** получить лист записи ЕГРИП и свидетельство ИНН (онлайн или в ФНС).
2. **День 2-3:** открыть расчётный счёт (для ЗАЗЕМЛИ — Точка, выполнено).
3. **День 3-5:** подключить эквайринг (онлайн-касса 54-ФЗ обязательна для D2C).
4. **День 3-5:** настроить ЭДО (для работы с Ozon — обязательно).
5. **День 5-7:** проверить автоматическую регистрацию в ПФР/ФСС (через личный кабинет ФНС). Если нет — подать заявление вручную.
6. **День 5-7:** настроить бухгалтерский сервис (Точка-бухгалтерия / Эльба / Моё Дело).
7. **День 7:** проверить, что письма от ФНС приходят на корректный e-mail.

---

## 6. Что НЕ применять

- **Не выбирать УСН 6%**, если расходная часть существенная (наш кейс).
- **Не игнорировать заявление на УСН** в первые 30 дней — иначе автоматически попадаешь на ОСН, что катастрофично для микробизнеса.
- **Не указывать только 1 ОКВЭД** — лучше сразу добавить смежные, чтобы потом не делать изменения.
- **Не открывать расчётный счёт до получения ОГРНИП** — банк всё равно попросит выписку.
- **Не путать ИНН физлица и ИНН ИП** — это один и тот же номер.
