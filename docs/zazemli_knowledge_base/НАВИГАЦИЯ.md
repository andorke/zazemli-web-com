## Принципы навигации

1. **Минто-структура** — от общего к частному.
2. **Полностью русский язык** — никаких atoms/molecules/foundations.
3. **Cross-refs в артефактах — короткие** (имя файла + ссылка на эту НАВИГАЦИЮ).
4. **При изменении структуры — правится только этот файл.**
5. **Lazy migration cross-refs.** Cross-refs со старыми путями работают через имя файла. Обновление — при следующей плановой правке артефакта.
6. **Единый источник биотопов 7 SKU.** 1-строки биотопов живут в `Продукт/leaflet-key-components-7-sku.md` v1.0 + дублируются в `Маркетинг/Каналы/Сайт/product-description.md` §«Биотопы по 7 SKU».

Развёрнутая форма: «Землю мы дали. Заботу — ты. Дневник — мост. Твоё растение скажет тебе спасибо.» Источник: `brand-voice-profile.md` v1.0.6 §Core formula.
