## Условия запуска (что нужно ДО bundle)

1. **Поставщик кашпо** — найти + договориться об опте. Кандидаты из live-аудита: Ломоносовская керамика (Я.Маркет общий бренд), Альба (Я.Маркет), местная Москва-керамика (для премиум-уровня).
2. **Дизайн-валидация** — кашпо должно соответствовать брендовой эстетике ZAZEMLI (Apothecary minimal, чёрный/bone/moss). Не любая керамика подойдёт.
3. **Логистика** — кашпо хрупкое, требует другой упаковки. COO-вопрос.
4. **Решение по партии 2 (через 6 нед после старта пилота)** — bundle добавляется в Сценарии «Зелёный» или «Жёлтый», не в «Красный».

## Тестовый объём (предложение)

При сценарии «Зелёный» партии 2 — **30–50 пилотных bundles** в общем объёме 500. Не основная ставка, валидация SKU.

Распределение по площадкам предположительно:
- Я.Маркет Москва: 20–30 bundles (премиум-аудитория, лучший ICP-fit для bundle)
- Ozon Москва: 10–20 bundles (валидация на массовой площадке)

## Открытые вопросы

1. **Какие SKU комплектовать с кашпо** — все 16 или только топ-5? Нужно решение CMO+CPO.
2. **Один универсальный кашпо или вариативность по растениям** (например, антуриуму керамика красная, монстере — moss-зелёная)?
3. **Поставщик кашпо** — сколько вариантов брать в рассмотрение? Нужны 3–5 коммерческих предложений.
4. **Лид-тайм поставщика** — для партии 2 (июль) нужно начать переговоры в мае–июне.
