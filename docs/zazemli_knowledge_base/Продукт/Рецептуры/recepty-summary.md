## Биотоп каждого SKU (1 строка)

- **N°001 Монстера** — влажные тропики Центральной Америки (Мексика–Гватемала); гемиэпифит-лиана, взбирается по стволу → нужны аэрация и опора.
- **N°002 Фикус** — тропики Юго-Восточной Азии; вечнозелёное дерево-гемиэпифит аллювиальных пойм → плотная база + структурный вес, без сфагнума.
- **N°003 Антуриум** — горные леса Анд (Колумбия–Эквадор); эпифит на коре → максимум аэрации, минимум торфа (10%).
- **N°004 Аглаонема** — тропики Юго-Восточной Азии; наземная теневая трава подлеска → лёгкая подкисленная база, толерантна.
- **N°005 Спатифиллум** — влажные тропики (Колумбия–Венесуэла); наземная розетка прибрежий → влагоёмко, но без застоя (аэренхима).
- **N°006 Замиокулькас** — сухая Восточная Африка; клубневый суккулент-геофит (CAM) → 50% минералки, без коры/сфагнума.
- **N°007 Эпипремнум** — тропики (эндемик Mo'orea, натурализован повсеместно); гемиэпифит-лиана → торф 30% + сфагнум, без песка.
