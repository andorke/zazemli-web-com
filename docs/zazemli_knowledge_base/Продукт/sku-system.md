Система SKU ЗАЗЕМЛИ — 16 SKU = 7 растений × 2-3 размера = «коллекция ритуалов».

## Что это

Документ, который отвечает: **из чего состоит наш продуктовый ассортимент, как организованы SKU, какова экономика по каждому**. Это продуктовый фундамент стратегии — на нём стоят все остальные операционные документы (Ozon-карточки, фото-бриф, IG-контент, дневник).

Не путать с физикой бокса (это — `05_production/box-layers.md`) и с маркетплейс-карточками (это — `05_production/marketplace-card-ozon.md`).

Краткая формула:
> **16 SKU = 7 растений × 2-3 размера. Каждый SKU = один ритуал заземления для конкретного растения. Каждое растение имеет фиксированный operational номер N°001-N°007 (см. §1).**

## Роль в ДНК-четвёрке (канал обучения)

Система SKU реализует все 4 источника ДНК через продукт:

| Источник ДНК | Как проявляется в SKU |
|---|---|
| **01 Natural-Botanical** | 7 родов растений с латынью (monstera, anthurium, spathiphyllum, zamioculcas, aglaonema, epipremnum, ficus) + 11 натуральных компонентов почвосмеси |
| **02 Apothecary** | Дозы по объёму горшка (1.2 / 2.2 / 3.5 л) + точные процентные пропорции компонентов + 5 г корневина + **операционная нумерация N°001-N°007** = аптекарская точность |
| **03 Naturalist** | Полное научное обоснование рецептур (80+ источников, привязка к биотопу) + дневник на год = наблюдение за циклом |
| **04 Handscript** | SKU-цвета как рукотворная палитра (6 цветов под 7 растений — N°001 Монстера и N°007 Эпипремнум делят цвет) |

Подробнее про источники — `02_voice/dna.md` §1.

**Формат записи (каноничный, везде одинаково):**
- Полная форма: `N°001 Монстера` (с пробелом после N° и неразрывным пробелом перед растением)
- Сокращённая (внутренние документы): `N001` или `№001`
- В именах файлов / технических ID: `n001-monstera` (kebab-case, латиница)
- В формулах URL: `?sku=001` или `/collectio/monstera` (название, не номер)

**Логика порядка нумерации (для команды и питча):**
- N°001 Монстера = флагман категории (узнаваемый род, образ бренда).
- N°002 Фикус = второй визуальный якорь (контрастный цвет, «дерево», эмоциональный фит).
- N°003-N°006 = по убыванию «образности»: Антуриум (цветок) → Аглаонема (декоративный лист) → Спатифиллум (декоративный цветок) → Замиокулькас (зелёная масса).
- N°007 Эпипремнум = замыкает коллекцию, делит цвет с N°001 (узор «коллекции»).

### 8. SKU как «коллекция ритуалов» (бренд-нарратив)

**Каждый SKU — не «вариант грунта», а отдельный ритуал заземления для конкретного растения.** Это критично для бренд-нарратива:
- Покупатель выбирает SKU **по своему растению** (Антуриум, Монстера), а не по литражу.
- 16 SKU вместе = «коллекция ритуалов» (отсюда же handle `@zazemli_collectio` + apothecary якорь COLLECTIO в брендбуке).
- Размер 1.2 / 2.2 / 3.5 л — это **«дозы ритуала»**, не «варианты товара».
- **N°001-N°007** — операционная нумерация = аптекарская подпись каждого ритуала. Покупатель видит N° на медальоне / дневнике / листовке — это как «номер серии» в premium-косметике.

Эта же логика идёт в:
- `mission.md` §1 «Эмоциональное ядро» — «через тактильный, эстетичный ритуал ухода за живым».
- `02_voice/dna.md` §3 — apothecary как ядро.
- IG-контент-план — каждый Reels работает на «один ритуал = одно растение = один N°».

### 9. Что продукт НЕ есть

- **Не магазин «всё для садоводства»** (узкий ассортимент — только 16 SKU, только 7 родов).
- **Не для опытных гроверов** — они смешают грунт сами дешевле. У нас другая ЦА.
- **Не массовый эконом** — только premium-сегмент.
- **Не «вечная коробка»** — коробку выбрасывают после пересадки, остаётся только дневник.
- **Не функциональный товар «грунт под растение»** — это категория HURY. Мы — wellness-ритуал.
- **Не подписка** на старте — V2-продукт через 12-24 мес (см. `mission.md` §4 Долгосрочная амбиция).
- **Не подарочный набор «общий»** — фокус на конкретное растение, не «набор для растениеводов».

## Правила применения

- **16 SKU = первая партия 500 боксов**. Расширение / сокращение ассортимента — только после оценки sell-through пилота Москва-only (см. `channels.md` §4).
- **Нумерация N°001-N°007 — каноничная и стабильная.** Не меняется при добавлении новых SKU — новые получают N°008+ (для V2-удобрений) или N°101+ (для V3-стартер-пака), и т.д.
- **Цены 1 890 / 2 190 / 2 590 ₽** — каноничные на старте (подтверждено Настей 2026-05-01). Изменение — только после оценки конкурентной выдачи через 3 месяца.
- **Размер выбирается по объёму горшка**, не по литражу грунта. Это критично для SEO-нейминга и копирайта.
- **Каждый SKU имеет свой цвет** (6 цветов на 7 SKU — N°001 и N°007 делят `#4A7C59`). Цвет используется в дудле, листовке, медальоне, видеообложке Ozon.
- **Рецептуры — каноничные**. Изменение % компонентов только после консультации с источниками `Заземли_научное_обоснование_рецептур.md` и Настей.
- **Дневник 212,73 ₽** = самая дорогая единичная позиция себестоимости. Оптимизация — открытый вопрос (см. §7).

## Что НЕ применять

- ❌ **Не вводить новые SKU** до оценки первой партии (500 боксов через пилот Москва-only). Гипотеза «больше SKU = больше продаж» в новой категории неверна — лучше точно знать спрос по 16 SKU.
- ❌ **Не делать «универсальный грунт»** или «грунт для всех растений» — это разрушает позиционирование «один ритуал = одно растение».
- ❌ **Не продавать отдельно** компоненты (дренаж, корневин, перчатки) — это разрушает bundle и переводит нас в категорию функционального товара.
- ❌ **Не использовать «1 575 ₽/литр»** как маркетинговый аргумент vs HURY — мы в другой категории, сравнение нерелевантно (см. `positioning.md` §3).
- ❌ **Не делать промо-скидки больше 10%** на старте — это размывает позиционирование. Используем bundle / подарок / эксклюзив, не скидку.
- ❌ **Не добавлять N° в название Ozon-карточки** — это шум для SEO и для покупателя в выдаче. N° — только для брендинга / физических артефактов / внутренних документов.
- ❌ **Не переставлять номера N°001-N°007** — они закреплены и используются в типографии (медальоны), дудлах, дневниках. Изменение разрушит уже напечатанные артефакты.
