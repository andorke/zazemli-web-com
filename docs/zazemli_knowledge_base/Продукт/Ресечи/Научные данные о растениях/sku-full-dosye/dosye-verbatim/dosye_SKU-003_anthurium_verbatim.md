## Указательные факты (без английского verbatim)

Эти факты в досье опираются на сводную логику или отсутствие данных, не на отдельную дословную цитату:

- **5.3 (досвет, watt/hours).** В S-источниках по виду конкретных watt/hour-рекомендаций для *A. andraeanum* нет; качественное обоснование строится через 1.3.1 (Chen, Li) + сезонная логика РФ. См. TODO 3.
- **8.2 (шаг увеличения горшка).** В NC State есть «every 2-3 years or rootbound», но числа «+2-3 см диаметра» нет. См. TODO 5.
- **9.2 (материал горшка).** Прямого verbatim отсутствует; косвенно — «Good Drainage» (NC State) + кислый субстрат + 60-80% влажности. См. TODO 6.
- **13.x (диагностическая таблица).** Компиляция из 3.2, 4.3, 11.2, 12.2, 16 — отдельной таблицы «симптом → причина» в S-источниках нет.
- **14.1 (акклиматизация).** В S-источниках по виду рекомендаций нет; опирается на Chen 2021 + общее правило NC State 12.2 «не допускать резких изменений».
- **19.1 NASA исходный список.** Сводный факт: антуриум **отсутствует** в первоначальном списке 12 видов Wolverton 1989 NASA-TM-101766; присутствует только в позднейшей сводной таблице Wikipedia с нулевыми показателями по бензолу/формальдегиду/TCE.
