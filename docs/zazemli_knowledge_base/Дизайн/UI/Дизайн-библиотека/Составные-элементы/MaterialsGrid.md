# MaterialsGrid · Версия: 1.0.0 · Дата: 2026-05-14 · Источник: DS v1.0 §5.4 + brand-v3 §15 (Apothecary description)

Молекула слоя `03_components/molecules/`. Сетка из 11 компонентов почвосмеси для страницы `/lab`. Каждый компонент = карточка с MaterialDot, именем, ролью, источником.

## Что это

Лабораторная решётка 2/3/4 колонок (mobile/tablet/desktop), показывающая 11 компонентов грунта. Каждая карточка — апотекарный протокол описания: материал-цвет (через MaterialDot), название, функциональная роль, ссылка-источник (peer-reviewed).

## Роль в ДНК-четвёрке

Мост **02 «Фармацевт» ↔ 03 «Натуралист»**:
- Структурная решётка + типографика — источник 02.
- MaterialDot (земляные тона) — источник 03 (материал как фактура).

Доля источников (по dna.md §7): ≈ 15% Галерист + 45% Фармацевт + 35% Натуралист + 5% Друг.

## Правила применения

1. **11 карточек ровно**, не 10 и не 12. Соответствует рецептурам Master Brief §6.5.
2. **Каждая карточка — точно один MaterialDot.** Не комбинировать материалы.
3. **Roles — императив с разделителем `·`.** Не запятая, не точка.
4. **Source — только peer-reviewed или `—`.** Если нет рецензируемого источника — прочерк, не вымышленная атрибуция.
5. **Bone-фон, тонкая charcoal-рамка opacity 0.15.** Никакого moss-фона (нарушит источник 01).
6. **Никаких SKU-цветов.** Это foundation-страница, не product page.

## Что НЕ применять

- Эмодзи в названии материала (например 🌱 рядом с «Биогумус»).
- Hover-эффекты внутри карточки (это статичный справочник, не интерактивная карта).
- Цветные плашки или рамки SKU-палитры.
- Маркетинговые описания («лучший дренаж в мире!» — anti-patterns.md §5.1).
