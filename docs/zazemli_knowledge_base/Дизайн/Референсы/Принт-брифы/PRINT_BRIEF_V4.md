# PRINT BRIEF · Заземли V5.4 · партия 0 (тестовая)

Pre-press спецификация для отправки в типографию.
Базовый принцип: **apothecary/гербарный материал**, uncoated дизайнерская бумага, без глянца.

> **v5.4 changelog:** добавлены 3 размера медальона (⌀80/100/120), мини унифицирована с медальоном, листовка переведена на uncoated лён без фоновой заливки, добавлены DIECUT-spot, K100 для текста, унифицированный safe zone.

| 007 | Эпипремнум | #4A7C59 (moss) | (=monstera) |

### 5.2 · Базовые цвета

| Роль | HEX | CMYK | Примечание |
|---|---|---|---|
| Bone (canvas, фон) | #F6F4F0 | C2 M2 Y4 K0 | Цвет бумаги ivory — не печатать на листовке |
| Charcoal ink (плашки) | #1C1C1C | C70 M67 Y66 K77 | Rich black для крупных плашек |
| **Charcoal text (внутри `<text>`)** | **#000000** | **K=100 pure** | См. §5.5 — split-rule для резкости текста на льне |
| Moss signature | #4A7C59 | C72 M32 Y75 K15 | |

### 5.3 · Safe zone (расстояние важных элементов от линии реза)

| Артефакт | Trim | Safe zone | Примечание |
|---|---|---|---|
| Медальон ⌀80/100/120 | round | **3 мм** | Стандарт ISO 12647-2 |
| Мини ⌀40 | round | **2 мм** | Компромисс для малых форматов · визуальная family с медальоном (94% ratio) > strict 3мм |
| Band 70×50 | rect | **3 мм** | f1 inset 9u от bleed-edge = 3мм от trim |
| Листовка 140×140 | rect | **3 мм** | f1 inset 18u от bleed-edge = 9u от trim |

### 5.4 · DIECUT layer (контур реза)

- **Spot color:** Pure Magenta 100% (CMYK `0 1 0 0`)
- **Layer name:** `CutContour` (стандарт adobe pre-press)
- **Position:** на bleed-edge (не на trim) — типография ориентируется на TrimBox в PDF metadata
- **Stroke:** 0.25 мм (0.75 SVG units)
- **Атрибут:** `data-spot="CutContour"` в SVG

### 5.5 · Charcoal split rule (новое)

Charcoal `#1C1C1C` имеет двойную интерпретацию в pre-press для оптимальной печати:

| Контекст | HEX в SVG | CMYK в PDF | Зачем |
|---|---|---|---|
| **Плашки, фоны, ground** | `#1C1C1C` | `C70 M67 Y66 K77` (rich black) | Глубокий насыщенный чёрный |
| **Текст внутри `<text>`** | автозамена → `#000000` | **`K=100 pure`** | Резкость на льне, нет misregistration |

Применяется в `pdf_cmyk()` post-process: внутри тегов `<text>` все `#1C1C1C` заменяются на `#000000` ДО ghostscript-конверсии. Rich black остаётся только на `<rect>`/`<circle>` плашках.

### 5.6 · Crop marks (метки реза)

- **Длина штриха:** 5 мм
- **Stroke:** 0.25 мм (K=100)
- **Gap от trim:** 2 мм (штрихи начинаются на 2 мм наружу от линии реза)
- **Позиции:** 8 штрихов (по 2 в каждом углу trim)

### 5.7 · Registration marks (приводочные кресты)

- **⌀ кружка:** 4 мм
- **Крест:** ±2 мм за края кружка (8 мм общий размер)
- **Stroke:** 0.25 мм (K=100)
- **Позиции:** 4 угла media-box (на расстоянии margin/2 от media-edge)

### 5.8 · PDF requirements

| Требование | Значение |
|---|---|
| Формат | **Vector PDF** (без растеризации) |
| Цветовое пространство | **CMYK FOGRA39L coated** (ICC profile) |
| Color conversion | sRGB → FOGRA39L через Ghostscript pdfwrite + post-process |
| Embedded fonts | **Subset** (Unbounded Regular/Bold/Black, Spectral Regular/Italic, Caveat Regular) с Cyrillic + Ё |
| TrimBox | Прописан в PDF metadata = реальный размер изделия |
| BleedBox | TrimBox + 3 мм со всех сторон |
| MediaBox | BleedBox + 7 мм margin (под crop+reg marks) |
| Версия PDF | 1.4 (для широкой совместимости) |

### 5.9 · Шрифты

- **Unbounded** 400/500/700/900 — для system text (caps labels, COLLECTIO header, hero name)
- **Spectral** Regular/Italic 400 — для Latin names + poetic signatures
- **Caveat** 400 — для handwritten tagline (только band + leaflet)

**⚠️ КРИТИЧНО:** Unbounded subset включает `Ё` (U+0401) — иначе «ЗАЗЕМЛЁН» на 6 SKU сломается. **Проверено на v5.4 — Ё embedded.**
