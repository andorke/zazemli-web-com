# 📁 Дизайн (CDO зона)

UI, печать, визуальные референсы. Как мы воплощаем бренд в интерфейсе и физических артефактах.

## 3 кита (Минто-структура)

| Папка | Зона | Что внутри |
|---|---|---|
| **UI/** | Веб-интерфейсы | Дизайн-библиотека (компоненты по уровням: Базовые-элементы / Составные-элементы / Секции-страниц / Шаблоны-страниц) + UX-стандарты |
| **Печать/** | Спека + production-ready | typography-print, prepress-workflow, print-brief, box-layers + `partia-0/` (финальные PDF для типографии) |
| **Референсы/** | Визуальные мокапы | Печатные (HTML-мокапы) + Принт-брифы + Упаковочные (PNG физических предметов) |

## Полная карта БЗ

→ См. **НАВИГАЦИЯ.md** в корне БЗ (`БЗ/НАВИГАЦИЯ.md`)

## Правила обновления

При добавлении / перемещении / переименовании артефактов внутри этой папки — **обновить `НАВИГАЦИЯ.md` в корне БЗ** (одна строка в соответствующей таблице).
