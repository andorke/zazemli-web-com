# box-layers · Версия: 1.0.0 · Дата: 2026-05-14 · Источник: brand-v3 §15 (Материальная архитектура)

Production-спека слоя `05_production/`. Физическая архитектура коробки: 3 формата, послойная сборка, инвентарь.

## Что это

Что лежит в коробке, в каком порядке, из чего состоит. Каждый слой — часть ритуала распаковки. **Это не компонент** — это physical-spec для сборки боксов на партии 0+.

## Роль в ДНК-четвёрке

Источник **03 Naturalist** (материалы, тактильность, фактура) + источник **02 Apothecary** (серийная нумерация уровней, каталогизация инвентаря). Физическое воплощение ритуала, который веб-organism `RitualSequence` описывает. См. `02_voice/dna.md` §1, §7.

## 3 формата коробки

| Формат | Размер мм | Объём | Цена | Растения |
|---|---|---|---|---|
| **S** | 200 × 200 × 100 | 1.2 л | 1 890 ₽ | суккуленты, молодые до 12 см |
| **M** · флагман | 250 × 250 × 100 | 2.2 л | 2 190 ₽ | антуриум, спатифиллум, аглаонема, замиокулькас |
| **L** | 320 × 220 × 100 (прямоугольная) | 3.5 л | 2 590 ₽ | монстера, фикус, большие эпипремнумы |

Крафт-картон без печати. На фронтальную стенку — круглая наклейка-медальон Ø 100 мм.

## Инвентарь · одной строкой

| Материал | Спека |
|---|---|
| Перчатки | нитриловые, чёрные, размер M. В конверте, уровень 04. |
| Корневин | 10 г, пакет «Лама-торф» + наклейка «Заземли рекомендует». |
| Пакет для дренажа | восьмишовный, белый матовый, без окошка. Керамзит. |
| Пакет для грунта | восьмишовный, белый матовый, с окошком. Состав виден. |
| Чёрный конверт | C5 155×105 мм, матовый. |
| Наполнитель | сотовая крафт-бумага (honeycomb kraft), чёрная или тёмно-крафтовая. |
| Крафт-коробка | 3 размера: S, M, L. Снаружи — медальон Ø 100 мм. |
| Мешочек (v2+) | лён 8×10 см, кулиска moss-шнуром, надпись + 00 ПРИХОД. |

## Правила сборки

1. Порядок укладки — **от дна к крышке**: коробка → наполнитель → дренажный пакет (02) → грунтовый пакет (03) → чёрный конверт (04) → листовка-гайд (05) → [v2+: мешочек (06)].
2. При открытии покупатель вынимает в обратном порядке — это и есть ритуал распаковки.
3. Каждый уровень с пакетом получает мини-стикер «01/02/03» в верхнем левом углу / под застёжкой.
4. Медальон Ø 100 мм — на фронтальную стенку коробки, по центру.
5. Структура: «один драматичный слой» = чёрная бумага + чёрный конверт + чёрные перчатки → chiaroscuro.

## Что НЕ применять

См. `02_voice/anti-patterns.md`:
- Зелёная бумажная стружка («WB-эстетика»).
- Бесцветный наполнитель / пенопласт.
- Цветные конверты (только чёрный матовый).
- Прозрачные пакеты для грунта / дренажа.
- Печатный брендинг на коробке (всё через наклейку-медальон).
- Стикеры в SKU-цвете (только charcoal на bone).
