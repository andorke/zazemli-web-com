## Параметры макета (зафиксированы)
- **Trim:** 148 × 195 мм. Figma-frame 420×553 px = 148:195 @72dpi → PDF-страница 420×553 pt = trim 1:1.
- **Вылет:** 3 мм со всех сторон → BleedBox **154 × 201 мм**. В макете вылета нет (фон белый, цвет даёт бумага) → вылет добавляется на препрессе расширением до 154×201 (MediaBox=BleedBox), TrimBox центрирован.
- **Скрепление:** скоба, 12 полос (кратно 4). Биговка фальцев — типография.
- **Поля:** симметричные 15,5/19,4 мм (скоба без отверстий — зеркальность не нужна).
- **Пунктирная trim-рамка (обводка фреймов) удалена** из макета во всех 84 полосах (2026-06-08) — печатается чистый край. Остаётся только внутренняя solid-рамка (дизайн).

## Цвет (РЕШЕНО)
- **Профиль: PSO Uncoated v3 / FOGRA52.** Выбран по A/B-цветопробе (FOGRA39 vs FOGRA52 рядом с листовкой) по инструкции EnjoyPrint. Файл профиля: `~/Documents/Claude/Projects/ЗАЗЕМЛИ/PSOuncoated_v3_FOGRA52.icc`.
- **PDF без внедрения профиля (untagged CMYK)** — требование Ольги (машина не делает двойную конверсию). Проверено: OutputIntent=0.
- **Charcoal / чёрный текст и хайрлайны → K=100** (нейтрально-тёмное ремапится в `0 0 0 1`; акценты не трогаются — у них канал выпадает).
- Прозрачности расфлэтчены (PDF 1.4). RGB-объектов в финале нет.

## Правки по итогам цветопробы (внесены в Figma 2026-06-08)
- **Дроби (¼/½) в календарях: кегль 8.5 → 9.5** (только глиф, текст рядом и интервал не тронуты). Затронуты Монстера(XI)/Фикус(XI)/Антуриум(V,VII,IX,XI)/Аглаонема(V,VII,IX)/Спатифиллум(V,VII,IX,XI). Замиокулькас/Эпипремнум — дробей в календарях нет. Итого 13 правок.
- **Аглаонема, стр XI — единственное цветовое исключение:** комментарии «фл:» (полив/подкормка/тёплый душ) + нижняя сноска «*фл…» → `#6B5C96` вместо SKU-iris. Кружки оставлены iris. (iris не прошёл по читаемости на мелком тексте, см. `sku_color_aa_rule`.)

## Пайплайн Figma → CMYK uncoated PDF (валидирован 2026-06-08)
Подробный рецепт — память `figma_pdf_prepress_pipeline`. Кратко:
1. **Figma → PDF постранично** (File → Export frames to PDF), 12 полос на SKU. Текст в кривые НЕ нужен — Figma экспортит Type3 с embed (= кривые).
2. **gs** RGB→CMYK FOGRA52, `-sColorConversionStrategy=CMYK -dOverrideICC -sOutputICCProfile=<FOGRA52> -dCompatibilityLevel=1.4`, untagged (без OutputIntent).
3. **K=100 remap** (qpdf uncompress + regex): нейтрально-тёмные `c m y k` → `0 0 0 1`.
4. **Вылет 3 мм + боксы** (pypdf): art на канвас 154×201 белым, TrimBox 148×195, Bleed/Media=154×201.
5. **Зачистка** DeviceRGB-групп прозрачности → DeviceCMYK.
6. **Сборка** 12 полос I→XII в один PDF на SKU.
