# print-brief · Версия: 1.0.0 · Дата: 2026-05-16 · Источник: PRINT_BRIEF v5.4 (26 апреля 2026) + MANIFEST.txt партии 0

Production-спека слоя `05_production/`. Каноническая pre-press спека партии 0 для отправки в типографию. 33 PDF, готовых к Enjoyprint. **Замена brand-v3 §16/§17.**

## Что это

Что печатаем, на каком материале, в каком тираже, с какими техническими настройками. 4 категории артефактов: листовка / медальоны / мини-стикеры / банд.

## Роль в ДНК-четвёрке

Источник **02 Apothecary** (sparse-тираж, серийная нумерация) + источник **03 Naturalist** (материалы: uncoated лён, soft-touch плёнка). См. `02_voice/dna.md` §1, §7.

## Калькулятор Enjoyprint · 4 категории

### 1 · ЛИСТОВКА 140×140 мм · 1 фронт + 7 оборотов

| Параметр | Значение |
|---|---|
| Формат | 140×140 мм |
| Вылеты | +3 мм · 146×146 мм с вылетом |
| Печать | Двусторонняя (фронт + SKU-оборот) |
| Бумага | **«Лен» сл. кость, 300 гр (uncoated)** |
| Ламинация / биговка / скругление | Не требуется |
| Срок | 3-5 рабочих дней |
| Пробный образец | 1000 ₽ — **взять** (проверить линии 0.25 мм на льне, hand-script читается) |
| Кол-во видов | 1 фронт + 7 оборотов = 8 PDF |
| Тираж партии 0 | 30-50 × 7 SKU = 210-350 шт |

**Файлы:** `leaflet-00-front.pdf` + `leaflet-00X-{sku}-back.pdf` × 7.

**Комментарии для типографии:**
- Печать **БЕЗ фоновой заливки** — бумага сама даёт цвет ivory (компенсирует риск белой полосы при сдвиге резки).
- Фронт универсальный (1 дизайн для всех 7 SKU).
- Оборот — 7 вариантов: имя растения + латынь + дудл + поэтика + состав + 2 QR + handscript в SKU-цвете.
- Тонкие линии 0.25 мм на льне — проверить пробным.

**Файлы:** `medallion-{80,100,120}mm-00X-{sku}.pdf` × 21.

**Комментарии:**

## Временной план

1. **Сейчас:** выбрать бумагу, оформить заказ на пробный образец листовки + медальона ⌀100 (1000 ₽).
2. **+3 дня:** получить образец, проверить линии 0.25 мм / шрифты / цвета на льне.
3. **+1 день:** подтвердить или скорректировать → полный заказ.
4. **+5 дней:** получить тираж.

## Что НЕ применять

См. `02_voice/anti-patterns.md` + PREPRESS_GUIDE anti-patterns:
- Bone-fill на листовке (бумага сама даёт цвет).
- Bone-fill только до trim для round (бытует белая полоса при сдвиге).
- `#1C1C1C` для текста (только rich black для плашек).
- DIECUT на bleed-edge (стандарт = trim).
- Скруглённые углы DIECUT (только если артефакт со скруглениями — band без скруглений!).
- Stroke <0.25 мм.
- Без TrimBox в metadata.
- RGB цвета (Enjoyprint отклонит).
- Шрифты без `uni=yes`.
