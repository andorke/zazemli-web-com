## 1.5 Decision gates

### Smoke-test → Wave 1

| Метрика | Зелёный | Жёлтый | Красный |
|---|---|---|---|
| Заказов за 7 дней | ≥ 2 | 1 | 0 |
| CR на показы | ≥ 0.5 % | 0.2–0.5 % | < 0.2 % |
| Доставка в срок | 100 % | 80–100 % | < 80 % |
| Жалобы на упаковку | 0 | 1 (не критическая) | 2+ или критическая |
| Отзывы | ≥ 4★ или 0 | 3★ | 1–2★ |

### Wave 1 (4 нед) → Решение по партии 2

| Метрика | Зелёный | Жёлтый | Красный |
|---|---|---|---|
| Продаж в сумме | ≥ 100 | 50–100 | < 50 |
| Рейтинг (n=10+) | ≥ 4.7 | 4.4–4.6 | < 4.4 |
| Возвраты | ≤ 8 % | 8–12 % | > 12 % |
| Я.Маркет CR vs Ozon | сравнимо или Я.Маркет лучше | Я.Маркет в 2× хуже | Я.Маркет в 5× хуже |
| СПб vs Москва | продаются | СПб медленнее | СПб не продаётся |
| Дневник упоминания | ≥ 5 % отзывов | 2–5 % | < 2 % |
| **CR(3.5 л) / CR(2.2 л)** (новое) | ≥ 0.5 | 0.3–0.5 | < 0.3 |

**Действие:**
- **Зелёный** → партия 2 = 500 боксов, опция B+ (3.5 л → 3 490 ₽), bundle pilot 30–50 шт, дневник как есть
- **Жёлтый** → партия 2 = 300 боксов, опция B без изменений, оптимизация дневника до 150 ₽, bundle отложить
- **Красный** → партия 2 = 200 боксов или пауза на pivot

### Противоречие #3 (новое в v1.1) · «Премиум выгорело» в MR v2.0 § 5.4

**MR v2.0 § 5.4** утверждает «слово премиум выгорело в категории». Live-аудит 2026-05-02 показал: **выгорело только на Ozon** (HURY и конкуренты), на **Я.Маркете слово работает** (Terracotta Plants Premium 5★, Профессиональный Premium, Премиум БиоБаза).

**Resolution:** позиционирование «curated, не премиум» применяется **только к Ozon-карточкам**. На Я.Маркет-карточках слово «премиум» используется. Это уточнение фиксируется в `pricing_decision` или в обновлении MR (требует sign-off, по правилу `Маркетинг/Ресеч рынка/CLAUDE.md`).
