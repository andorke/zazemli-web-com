## Финансы / Операционка

- **[2026-06-07] Контроль входящей коры (aged/composted)** — ввести в приёмку сырья обязательную проверку, что кора выдержанная/компостированная, а не свежая. Свежая кора иммобилизует 60–120 мг N/л за 28 дней (Jackson 2008) → растение первый месяц голодает по азоту. В листе «Закупки» кора — orchid bark bioabsolut, нужно подтвердить статус у поставщика. Роль: COO. Источник: СРО-аудит §Риск 2.

- **[2026-06-07] Контрольный замер pH/EC на партию каждого SKU** — дешёвым pH/EC-метром, один замер на партию. Переводит параметры рецептур из расчётных в измеренные → усиление claim и юр-чистоты (слоган «Эмпирика надёжнее»). Роль: CPO/COO. Источник: СРО-аудит §Риск 4.

- **[2026-06-07] Просчёт фактического веса готовых боксов Замиокулькас/Фикус** — минеральная фракция 50%/35% + тяжёлый цеолит делают эти SKU существенно тяжелее ароидных. Взвесить готовые боксы, заложить вес в тариф Ozon (зависит от веса) и юнит-экономику. Для фикуса тяжесть — плюс (устойчивость кроны), для логистики — статья затрат. Роль: CFO/COO. Источник: СРО-аудит §Риск 3.

- **[2026-05-13] Бюджет печати дневников — per-SKU vs per-штука** — для монстеры партия 0 (550 шт) обошлась в 117 000 ₽ (= 213 ₽/шт). До массового тиража уточнить с типографией: при заказе 6 новых SKU тираж per-SKU будет меньше — растёт ли цена за штуку из-за приладки? Нужно для расчёта юнит-экономики Collectio до запуска 2026-Q3. Источник: чат 2026-05-13, задача `2026-05_dnevniki-6-SKU`. Связано: `/Рабочая папка/2026-05_dnevniki-6-SKU/README.md`, БЗ «Юнит-экономика».
