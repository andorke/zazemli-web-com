## Принципы (общие для всех changes)

- **Source of truth**: вёрстка — прототип страницы; текст — канон-`.md`; при расхождении текста побеждает канон. **Два исключения, где канон заведомо отстаёт от прототипа**: (1) /lab — и вёрстка, и текст из `lab.html` (канон v2.0 отстаёт, зафиксировано в vault); (2) founder-quote на страницах товара — новая редакция «…а руками тянется к земле» из прототипов (канон-патч `product-description.md` pending на стороне vault).
- Прототипы имеют один брейкпоинт 860px — адаптив до 360px достраиваем по DS-правилам.
- Каждый change оставляет сайт рабочим: static export зелёный, тесты зелёные.
- Находки, требующие решения фаундера, копятся в файл «вопросы Насте» — не блокируют вёрстку.

## Дизайн (одобрен по секциям)

### Change 1 — `ds-migration` (блокирует остальные)

Токены `globals.css` → tokens.json v1.1.0: `--moss-ink #406C4F`, `chalk`, earth-палитра, 7 SKU-цветов; семантика: текст/ссылки/латынь на bone = `moss-ink`/charcoal, raw `moss` — только заливки и крупное ≥18pt. Типо/spacing-шкала (роли 84/52/34/24/18/15/13/12 + space-блок) → CSS-переменные. Шрифты: Unbounded/Spectral/Caveat выводятся; **Newsreader** (300/400/500 + italic) и **Commissioner** (400/500) — self-hosted woff2 с субсеттингом (кириллица+латиница) через `next/font/local`, по текущему пайплайну. Общие компоненты по прототипам: topbar (Коллекция · Лаборатория · Гайд + бургер <860), footer (соцсети, email, дисклеймер), `<details>`-аккордеон, MaterialDot, кнопки btn/btn--solid. Переходное состояние: font-переменные переименовываются сразу во всём коде — старые секции выглядят «переходно» до change 2 (осознанный трейд-офф, «3 шрифта max»). ds-lint обновляется: запрет raw moss на мелком тексте, запрет старых font-переменных.

### Change 2 — `landing-redesign`

Главная по `landing.html` (12 секций): шапка → Hero → манифест → «Три шага» (новая) → галерея 7 SKU → «что в боксе» → колбы → «что даёт» → о нас → тизеры → Ozon-CTA («От 1 890 ₽») → футер. Тексты — `home.md` v2.1 (Hero H1 «Заземли растение. Заземли себя.»). Тихий мост «почему именно так →» — ссылка `moss-ink` на реальный роут `/lab` (якорь `#lab` в прототипе — артефакт одностраничности). Карточки галереи становятся кликабельными → `/collectio/[растение]` (дефолт; вопрос Насте). Секции правятся по одной, `image-slot.tsx` остаётся механизмом плейсхолдеров `.svis`. `src/content/home.ts` расширяется; инварианты тестов сохраняются.

### Change 3 — `product-pages`

Новый роут `/collectio/[slug]` + `generateStaticParams` на 7 slug'ов (monstera, ficus, anthurium, aglaonema, spathiphyllum, zamioculcas, epipremnum). Единый шаблон по `collectio-*.html`: 8 блоков — hero (SKU-цвет как декор) → опись → колба-схема % → состав с MaterialDot → SourceNote (`<details>`, evidence-base, без внутряка) → care-блок с дневник-панелью → размер-селектор → Ozon-CTA. SKU-цвет — только декор; текст/кнопки `moss-ink`/charcoal; «never 2+ SKU colors on one page» — кандидат в ds-lint. `/collectio` (голый): страница-заглушка → клиентский редирект (meta refresh + `location.replace('/#collectio')` + canonical на `/`). «Caveat»-акцент в прототипах = Newsreader italic + SKU-цвет (не шрифт Caveat) — так и верстаем. Нумерация на страницах товара — N°001 (N°01 — только /lab, как в прототипах).

**/guide** по `guide.html` + `guide.md` v3.0 (совпадают дословно): 3 тира, Тир-3 = CJM 5 стадий 00–04 (Подготовка · Конверт · Дренаж · Грунт · Забота), маркировка ● «в боксе» / ○ «своё», стадия = «Что понадобится → Шаги → Подсказка → Готово, когда», Schema.org HowTo, Caveat 0. **/lab** по `lab.html` (единственная страница, где прототип = источник и вёрстки, и текста): HERO → проблема (якорь: аэрация — лимитирующий фактор) → 11 компонентов (аккордеон 4 группы, нарративы + source-notes с DOI) → 7 рецептур N°01–07 (биотоп + MaterialDot) → «что земля даёт» → «Эмпирика надёжнее» (центрирован) → «чем это не универсальный грунт» → список литературы → Ozon-CTA → дисклеймер в футере /lab.
